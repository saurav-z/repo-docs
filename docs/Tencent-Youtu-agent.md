Youtu-Agent is a versatile, high-performance framework designed for building autonomous agents that excel in various tasks, including data analysis, file processing, and research, all powered by open-source models. This framework offers a streamlined approach to agent development, emphasizing ease of use, flexibility, and cost-effectiveness.

## ‚ú® Key Features

*   **High Performance & Strong Benchmarks:** Achieves state-of-the-art results on benchmarks like WebWalkerQA (71.47% pass@1) and GAIA (72.8% pass@1 on the text-only subset) using DeepSeek-V3 series models, demonstrating its effectiveness.
*   **Open-Source Focused & Cost-Effective:** Designed for accessible, low-cost deployment, eliminating reliance on closed-source models.
*   **Versatile Use Cases:** Out-of-the-box support for tasks like CSV analysis, literature reviews, personal file organization, and more, with podcast and video generation coming soon.
*   **Flexible & Extensible Architecture:** Built on [openai-agents](https://github.com/openai/openai-agents-python), supporting diverse model APIs (DeepSeek, gpt-oss, etc.), tool integrations, and framework implementations.
*   **Simplified Development:** YAML-based configuration, automatic agent generation, and streamlined setup minimize manual effort.
*   **Automatic Agent Generation:** Quickly prototype agents using simple, YAML-based configurations generated by the system itself.

## ü§î Why Choose Youtu-Agent?

Youtu-Agent provides significant value for various user groups:

*   **For Agents Researchers & LLM Trainers:** Offers a strong baseline for model training and ablation studies, and one-click evaluation scripts.
*   **For Agent Application Developers:** Provides a proven and portable scaffolding for real-world agent applications with a rich set of built-in toolkits.
*   **For AI & Agent Enthusiasts:** Features practical use cases, simplicity, and debugging tools for intuitive development.

## üß© Core Concepts

*   **Agent:** An LLM configured with specific prompts, tools, and an environment.
*   **Toolkit:** An encapsulated set of tools an agent can use.
*   **Environment:** The world in which the agent operates.
*   **ContextManager:** Module for managing the agent's context window.
*   **Benchmark:** Encapsulated workflow for a specific dataset.

Refer to our [technical documentation](https://tencentcloudadp.github.io/youtu-agent/) for implementation details.

## üöÄ Getting Started

Get started quickly with Youtu-Agent using the following steps.

### Setup

#### Source Code Deployment

```bash
git clone https://github.com/TencentCloudADP/youtu-agent.git
cd youtu-agent
uv sync  # or, `make sync`
source ./.venv/bin/activate
cp .env.example .env  # Fill in your API keys!
```

Fill in the necessary API keys in the `.env` file.  Example:

```bash
# llm requires OpenAI API format compatibility
# setup your LLM config , ref https://api-docs.deepseek.com/
UTU_LLM_TYPE=chat.completions
UTU_LLM_MODEL=deepseek-chat
UTU_LLM_BASE_URL=https://api.deepseek.com/v1
UTU_LLM_API_KEY=replace-to-your-api-key
```

*   **DeepSeek API Free Tokens:** [Tencent Cloud International](https://www.tencentcloud.com/) offers new users of the DeepSeek API **3 million free tokens** (**Sep 1 ‚Äì Oct 31, 2025**). [Try it out](https://www.tencentcloud.com/document/product/1255/70381) for free. Replace the API key in the .env file below:

```bash
# llm
# setup your LLM config , ref https://www.tencentcloud.com/document/product/1255/70381
UTU_LLM_TYPE=chat.completions
UTU_LLM_MODEL=deepseek-v3
UTU_LLM_BASE_URL=https://api.lkeap.cloud.tencent.com/v1
UTU_LLM_API_KEY=replace-with-your-api-key
```

#### Docker Deployment

See [`docker/README.md`](./docker/README.md) for a streamlined Docker-based setup with an interactive frontend.

### Quick Start

Run an interactive CLI chatbot with a simple search tool agent:

```bash
# NOTE: You need to set `SERPER_API_KEY` and `JINA_API_KEY` in `.env` for web search access.
# (We plan to replace these with free alternatives in the future)
python scripts/cli_chat.py --stream --config default
# To avoid using the search toolkit, you can run:
python scripts/cli_chat.py --stream --config base
```

### Run Evaluations

Evaluate on `WebWalkerQA`:

```bash
# Prepare dataset. This script will download and process WebWalkerQA dataset, and save it to DB.
python scripts/data/process_web_walker_qa.py

# Run evaluation with config `ww.yaml` with your custom `exp_id`. We choose the sampled small dataset `WebWalkerQA_15` for quick evaluation.
# NOTE: `JUDGE_LLM_TYPE, JUDGE_LLM_MODEL, JUDGE_LLM_BASE_URL, JUDGE_LLM_API_KEY` should be set in `.env`. Ref `.env.full`.
python scripts/run_eval.py --config_name ww --exp_id <your_exp_id> --dataset WebWalkerQA_15 --concurrency 5
```

Results are stored and can be analyzed in the evaluation platform.
