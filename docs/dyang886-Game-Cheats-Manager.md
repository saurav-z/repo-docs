## Key Features

*   **Effortless Trainer Management:** Organize and manage your game trainers from a single, intuitive interface.
*   **Trainer Browsing & Downloading:** Browse, search, and download trainers directly within the application.
*   **One-Click Launch:** Launch trainers with a simple click of a button.
*   **Integrated Trainer Management:** Manage trainer sources, update settings, and customize your experience.
*   **Easy Installation:**  Download and install the application to begin using the trainers.

## How to Use

1.  **Browse Trainers:** Use the search bar or browse the list in the left column to find downloaded trainers. Launch or delete trainers directly.
2.  **Download Trainers:** Search in the right column using keywords, and double-click a desired match to download it.
3.  **Trainer Management:** Configure settings for each trainer source, including automatic updates, trainer data settings, and download servers.
4.  **Options:**
    *   Customize settings such as themes and languages.
    *   Import trainers from files.
    *   Open the trainer download folder.
    *   Add the trainer download path to the Windows Defender whitelist.
    *   View app version and project-related links in the "About" section.

## Installation Guide

1.  **Download:** Get the latest Windows (64-bit) installer from the [releases page](https://github.com/dyang886/Game-Cheats-Manager/releases).
2.  **Run:** Execute the installer and follow the on-screen instructions.
3.  **Launch:** Open Game Cheats Manager from your applications folder or start menu.
