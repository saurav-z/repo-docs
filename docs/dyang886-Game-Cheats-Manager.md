## Key Features

*   **Effortless Trainer Management:**  Browse, download, launch, and delete trainers all from a single, user-friendly interface.
*   **Integrated Trainer Search:** Quickly find the trainers you need with a built-in search function that accesses a large database of available trainers.
*   **Direct Launch Functionality:**  Launch trainers directly from within the application for a seamless gaming experience.
*   **Customizable Settings:**  Adjust themes, languages, and other settings to personalize your experience.
*   **Trainer Management Panel:** Manage all the settings for each trainer source, including automatic updates, trainer data, and download server selection.
*   **Import/Export Options:**  Easily import existing trainers and manage your library.
*   **Windows Defender Whitelisting:**  Add the trainer download path to the Windows Defender whitelist directly within the application.

## Getting Started

### Installation

1.  **Download:**  Get the latest Windows (64-bit) installer from the [latest releases page](https://github.com/dyang886/Game-Cheats-Manager/releases).
2.  **Run:**  Execute the installer and follow the on-screen instructions.
3.  **Launch:**  Open Game Cheats Manager from your applications folder or the Start menu.

### Usage Overview

1.  **Browse/Launch Trainers:** Use the search bar in the left column to find trainers, then double-click or click the "Launch" button.
2.  **Download Trainers:** Search for trainers in the right column and double-click to download.
3.  **Trainer Settings:** Access the Trainer Management panel to adjust settings and configurations.
4.  **Options Menu:** Use the "Options" menu for settings, import, and other helpful functionalities.
