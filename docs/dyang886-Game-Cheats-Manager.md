## Key Features

*   **Easy Trainer Browsing:** Quickly search and locate trainers for your favorite single-player games.
*   **One-Click Downloads:** Download trainers directly within the application.
*   **Seamless Trainer Launching:** Launch downloaded trainers with a simple click.
*   **Effortless Trainer Management:** Organize and delete trainers from a centralized location.
*   **Customization Options:** Adjust settings, import trainers, and manage download paths.
*   **Automatic Updates:** Stay up-to-date with the latest trainer versions.

## How to Use Game Cheats Manager

1.  **Browse & Launch Trainers:** Use the search bar or browse the list on the left to find your downloaded trainers. Double-click or click "Launch" to start a trainer.
2.  **Download Trainers:** Search for trainers using keywords in the right column and press Enter. Double-click the desired result to download it. Customize the download path by clicking the "..." button.
3.  **Trainer Management:** Access the Trainer Management panel to configure settings like auto-updates, trainer source preferences, and download server.
4.  **Options Menu:**
    *   **Settings:** Customize your theme and language preferences.
    *   **Import Trainers:** Import trainers from your file system.  Note: Imported trainers cannot auto-update.
    *   **Open Trainer Download Path:** Quickly access your trainer download folder.
    *   **Add Paths to Whitelist:** Add the trainer download path to your Windows Defender whitelist.
    *   **About:** View application version and project-related links.

## Installation

1.  **Download:** Get the latest Windows (64-bit) installer from the [latest release page](https://github.com/dyang886/Game-Cheats-Manager/releases).
2.  **Run Installer:** Execute the downloaded file and follow the on-screen instructions.
3.  **Launch:** Open Game Cheats Manager from your applications folder or start menu.
