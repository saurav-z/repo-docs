## Key Features

*   **Effortless Trainer Browsing & Downloading:** Search and download trainers directly within the application.
*   **Simplified Trainer Management:** Launch and delete trainers with a single click.
*   **Centralized Hub:** Manage all your trainers from one convenient location.
*   **Customizable Settings:** Configure trainer sources, update settings, and more.
*   **Easy Installation:** Simple, straightforward installation process.

## How to Use

1.  **Browse/Launch Trainers:** Use the left-hand search to locate downloaded trainers, then launch or delete them via buttons.
2.  **Download Trainers:** Use the right-hand search to discover trainers and download them.  Customize the download path.
3.  **Trainer Management:** Adjust settings for individual trainer sources, including automatic updates.
4.  **Options Menu:** Access settings, import trainers, open download folder, add paths to the Windows Defender whitelist, and view app information.

## Installation Guide

1.  **Download:** Get the latest installer from the [latest release](https://github.com/dyang886/Game-Cheats-Manager/releases) (Windows 64-bit).
2.  **Run:** Execute the installer and follow the on-screen prompts.
3.  **Launch:** Open Game Cheats Manager from your applications folder or start menu.
