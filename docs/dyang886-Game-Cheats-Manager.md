## Key Features

*   **Effortless Trainer Management:** Browse, download, launch, and delete trainers all from a single, user-friendly interface.
*   **Integrated Trainer Search:** Quickly find trainers for your favorite single-player games using keywords.
*   **Customizable Download Paths:**  Easily configure where your trainers are stored.
*   **Trainer Management Panel**: Settings for each trainer source, like automatic updates.
*   **Essential Options:** Access settings for themes, languages, import trainers, open download paths, and add paths to the Windows Defender whitelist.

## How to Use

1.  **Browse & Launch Trainers:** Find your installed trainers in the left column.  Double-click or use the "Launch" button to start a trainer, or "Delete" to remove it.
2.  **Download Trainers:** Search for trainers in the right column using keywords and press Enter to find trainers to download. Double-click the desired trainer to download. The download path can be changed.
3.  **Trainer Management:** The `Trainer Management` panel provides settings for each trainer source, such as automatically updating trainers, or changing download servers.
4.  **Options Menu:** Access settings and additional functionality via the options menu:
    *   **Settings:** Customize themes and languages.
    *   **Import Trainers:** Add existing trainers from your system.
    *   **Open Trainer Download Path:** Easily access your trainer folder.
    *   **Add Paths to Whitelist:**  Add your trainer download path to Windows Defender's whitelist.
    *   **About:** View application version and project-related links.

## Installation

1.  **Download the Installer:** Get the latest installer from the [latest release page](https://github.com/dyang886/Game-Cheats-Manager/releases) for Windows (64-bit).
2.  **Run the Installer:**  Execute the downloaded file and follow the on-screen instructions to install the application.
3.  **Launch the Application:** Open Game Cheats Manager from your applications folder or start menu.
