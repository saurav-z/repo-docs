## Key Features

*   **Browse and Launch Trainers:** Easily launch your downloaded trainers with a single click.
*   **Integrated Trainer Search and Download:**  Quickly find and download trainers directly within the application.
*   **Trainer Management:** Manage trainer sources, auto-updates, and download locations.
*   **Customization Options:**  Personalize your experience with theme and language settings.
*   **Import & Whitelist:** Import existing trainers and add download paths to the Windows Defender whitelist for seamless operation.
*   **One-Stop Solution:** Manage all your trainers from one convenient location, keeping your gaming experience organized and accessible.

## How to Use

1.  **Browse Trainers:** Use the search bar or browse the left column to find downloaded trainers, then launch or delete.
2.  **Download Trainers:** Search for trainers in the right column and double-click to download. Customize download path if needed.
3.  **Trainer Management:** Configure settings for trainer sources, auto-updates, and server settings.
4.  **Options:** Access settings, import trainers, open download paths, add paths to the whitelist, and view app information.

## Installation

1.  **Download:** Download the installer from the [latest release](https://github.com/dyang886/Game-Cheats-Manager/releases).
2.  **Run Installer:** Execute the downloaded file and follow the on-screen prompts.
3.  **Launch:** Open Game Cheats Manager from your applications folder or start menu.
