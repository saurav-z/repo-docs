## Key Features:

*   **Browse and Launch Trainers:** Easily browse your downloaded trainers and launch them directly with a simple click.
*   **Download Trainers:** Search for and download new trainers directly within the application.
*   **Trainer Management:** Configure settings such as automatic trainer updates and source management from a single place.
*   **Customizable Options:** Personalize your experience with theme and language settings, import trainers, and manage download paths.
*   **Windows Defender Whitelist:** Easily add the trainer download path to the Windows Defender whitelist for seamless operation.

## How to Use:

1.  **Browse Trainers:** Use the search bar or browse the list to find downloaded trainers. Double-click or click `Launch` to run them. Click `Delete` to remove.
2.  **Download Trainers:** Search using keywords and press `Enter` to find available trainers. Double-click to download them. The download path is shown and can be changed.
3.  **Trainer Management:** Adjust settings for trainer sources, including auto-updates and data.
4.  **Options Menu:**
    *   **Settings:** Adjust themes and languages.
    *   **Import Trainers:** Import trainers from your file system. Imported trainers do not auto-update.
    *   **Open Trainer Download Path:** Quickly access your download folder.
    *   **Add Paths to Whitelist:** Whitelist your download path in Windows Defender.
    *   **About:** View application version and project links.

## Installation:

1.  **Download:** Get the latest Windows (64-bit) installer from the [releases page](https://github.com/dyang886/Game-Cheats-Manager/releases).
2.  **Run Installer:** Execute the downloaded file and follow the on-screen prompts.
3.  **Launch:** Open Game Cheats Manager from your applications folder or Start Menu.
