# Game Cheats Manager: Your Ultimate Trainer Management Solution

Tired of juggling multiple trainer applications for your favorite single-player games? **Game Cheats Manager** is a user-friendly application designed to streamline your gaming experience by centralizing the management, downloading, and launching of trainers, all in one convenient place. Explore the [original repository](https://github.com/dyang886/Game-Cheats-Manager) for more details and to contribute.

## Key Features

*   **Centralized Trainer Management:** Easily browse, download, and launch all your trainers from a single application.
*   **Trainer Browsing & Searching:** Quickly find trainers by searching keywords.
*   **Direct Launch & Delete:** Launch or delete trainers directly from the application.
*   **Trainer Download Management:** Download trainers from various sources within the app.
*   **Customizable Settings:** Adjust settings for trainer sources, automatic updates, download servers, themes, and languages.
*   **Import Trainers:** Import existing trainers for unified management.
*   **Windows Defender Whitelist Integration:** Add trainer download paths to the Windows Defender whitelist for seamless functionality.

## How to Use

1.  **Browse & Launch Trainers:** Navigate trainers in the left column. Double-click or click "Launch" to start a trainer, or "Delete" to remove it.
2.  **Download Trainers:** Use the search bar in the right column to find and download new trainers.
3.  **Manage Trainers:** Use the `Trainer Management` panel to adjust the settings for each trainer source, like enabling automatic updates and changing the download server.
4.  **Customize with Options:** Access settings for themes, languages, trainer imports, open the trainer download path, add paths to the whitelist, and access the "About" information.

## Installation

1.  **Download:** Get the latest Windows (64-bit) installer from the [releases page](https://github.com/dyang886/Game-Cheats-Manager/releases).
2.  **Run:** Execute the downloaded installer and follow the on-screen instructions.
3.  **Launch:** Open Game Cheats Manager from your applications folder or start menu.
