## Key Features

*   **Unified Trainer Management:** Browse, download, launch, and delete trainers all from one easy-to-use interface.
*   **Trainer Browsing & Downloading:**  Search for trainers using keywords and download them directly within the application.
*   **Flexible Trainer Management:** Control trainer updates, search data, and download server settings.
*   **Customizable Settings:**  Adjust themes, languages, and import your existing trainers.
*   **Windows Defender Whitelist:** Easily add your trainer download path for smoother gameplay.

## How to Use

1.  **Browse & Launch Trainers:**  Use the left panel to search and launch downloaded trainers with a double-click or by pressing the `Launch` button. Delete them with the `Delete` button.
2.  **Download Trainers:** Search for trainers in the right panel and download them with a double-click.  Adjust the download path via the `...` button.
3.  **Trainer Management Panel:** Access advanced settings for each trainer source, including auto-update and download server configurations.
4.  **Options Menu:** Customize your experience with settings for themes, languages, trainer import, and more, including opening the trainer download folder and adding the download path to the Windows Defender whitelist.

## Installation

1.  **Download:** Get the latest Windows (64-bit) installer from the [latest release page](https://github.com/dyang886/Game-Cheats-Manager/releases).
2.  **Run Installer:** Execute the downloaded installer and follow the on-screen instructions.
3.  **Launch:** Open Game Cheats Manager from your applications folder or start menu.
