## Key Features

*   **Intuitive Trainer Browsing:** Easily search and browse trainers for your favorite single-player games.
*   **Seamless Trainer Downloading:** Download trainers directly within the application.
*   **Efficient Trainer Management:** Launch, delete, and manage your trainers from a centralized location.
*   **Customizable Settings:** Adjust themes, languages, and other preferences to personalize your experience.
*   **Trainer Management Panel:** Configure settings for auto-updating and trainer search data.
*   **Import Trainers:** Easily import trainers from your existing collection.
*   **Whitelist Support:** Add the trainer download path to your Windows Defender whitelist.

## How to Use Game Cheats Manager

1.  **Browse Trainers:** Use the search bar or browse the list on the left to find installed trainers. Double-click or click `Launch` to start a trainer, or `Delete` to remove it.
2.  **Download Trainers:** Search for trainers by keyword in the right column and press `Enter`. Double-click a match to download. Customize the download path by clicking `...`.
3.  **Trainer Management Panel:** Access advanced settings for your trainer sources, including auto-updates and search data.
4.  **Options Menu:** Access settings such as themes, language, trainer import, and access the trainer download path. Add the trainer download path to the Windows Defender whitelist.

## Installation

1.  **Download:** Download the 64-bit installer from the [latest release](https://github.com/dyang886/Game-Cheats-Manager/releases).
2.  **Run Installer:** Execute the downloaded file and follow the on-screen instructions.
3.  **Launch:** Open Game Cheats Manager from your applications folder or start menu.
