## Key Features

*   **Browse and Search Trainers:** Easily find trainers for your favorite single-player games with a built-in search function.
*   **Download Trainers Directly:** Download trainers directly from the app with a simple double-click.
*   **Effortless Trainer Management:** Launch and delete trainers with a click, keeping your gaming experience organized.
*   **Customizable Settings:** Tailor your experience with options for themes, languages, and trainer source management.
*   **Import & Auto-Update Options:** Import existing trainers, plus options for auto-updating to keep your trainers current.
*   **Whitelist Integration:** Seamlessly add trainer paths to Windows Defender for hassle-free usage.

## How to Use

1.  **Browse Trainers:** Use the search bar or browse the left column to find downloaded trainers. Launch or delete trainers with the respective buttons.
2.  **Download Trainers:** Search for trainers in the right column. Double-click to download. Customize your download path by clicking the "..." button.
3.  **Trainer Management:** Manage trainer sources, update settings, and more within the "Trainer Management" panel.
4.  **Options Menu:** Access settings, import trainers, open the trainer download path, add paths to the Windows Defender whitelist, and view app information.

## Installation

1.  **Download:** Download the installer for Windows (64-bit) from the [latest release](https://github.com/dyang886/Game-Cheats-Manager/releases).
2.  **Run Installer:** Execute the downloaded file and follow the on-screen instructions.
3.  **Launch:** Open Game Cheats Manager from your applications folder or start menu.
