## Overview

Open-LLM-VTuber is an innovative project that provides you with a voice-interactive AI companion featuring a Live2D avatar, supporting real-time voice conversation, visual perception, and much more, all running on your local machine. This project is perfect for creating a virtual girlfriend, boyfriend, pet, or any character you desire, offering both web and desktop client modes, with a special transparent background desktop pet mode.

**Key Features:**

*   **Cross-Platform Compatibility:** Works flawlessly on Windows, macOS, and Linux.
*   **Offline Mode:** Run completely offline using local models, ensuring privacy and security.
*   **Interactive Web & Desktop Clients:** Offers flexible modes, including a desktop pet mode for an ever-present AI companion.
*   **Advanced Interaction:**
    *   Visual perception (camera, screen recording, screenshots)
    *   Microphone voice interruption
    *   Touch feedback
    *   Live2D expression
    *   Pet mode with transparent background
    *   AI's inner thoughts, display expressions, thoughts and actions without them being spoken
    *   AI proactive speaking feature
    *   Chat log persistence
    *   TTS translation support (e.g., chat in Chinese while AI uses Japanese voice)
*   **Extensive Model Support:** Wide range of LLMs, ASR, and TTS models supported.
*   **Highly Customizable:** Configure and customize your AI companion's appearance and persona using the character customization guide.
