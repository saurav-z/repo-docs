## üöÄ Key Features

*   **Cross-Platform Compatibility**: Seamlessly runs on Windows, macOS, and Linux, supporting both NVIDIA and non-NVIDIA GPUs, with CPU fallback options.
*   **Offline Mode**: Enjoy complete privacy and security with fully offline functionality using local models ‚Äì no internet connection needed.
*   **Web and Desktop Clients**: Offers both web and desktop client options, providing rich interactive features and personalized settings, including a unique desktop pet mode.
*   **Advanced Interaction**:
    *   Visual perception via camera, screen recording, and screenshots.
    *   Voice interruption and touch feedback.
    *   Live2D expression control and a unique pet mode with a transparent background.
    *   Display of AI's inner thoughts and AI proactive speaking.
    *   Persistent chat logs for continuous conversations.
    *   TTS translation support.
*   **Extensive Model Support**:
    *   **LLMs**:  Ollama, OpenAI, Gemini, Claude, Mistral, DeepSeek, Zhipu AI, GGUF, LM Studio, vLLM, and more.
    *   **ASR**: sherpa-onnx, FunASR, Faster-Whisper, Whisper.cpp, Whisper, Groq Whisper, Azure ASR, etc.
    *   **TTS**: sherpa-onnx, pyttsx3, MeloTTS, Coqui-TTS, GPTSoVITS, Bark, CosyVoice, Edge TTS, Fish Audio, Azure TTS, etc.
*   **Highly Customizable**:
    *   Easy module configuration.
    *   Character customization with Live2D models, prompts, and voice cloning.
    *   Flexible Agent implementation.
    *   Modular design for easy extensibility.

## ‚ú® What is Open-LLM-VTuber?

Open-LLM-VTuber is a cutting-edge project bringing your ideal AI companion to life. This innovative AI-powered virtual entity offers real-time voice interaction, visual awareness, and a captivating Live2D avatar that you can fully customize. Create your perfect virtual girlfriend, boyfriend, or companion, and enjoy rich interactions, all while keeping your conversations private and secure by running it entirely offline on your computer.
