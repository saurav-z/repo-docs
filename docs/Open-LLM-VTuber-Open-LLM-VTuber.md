# Open-LLM-VTuber: Your AI Companion with a Voice and a Face 

> Create your own interactive, AI-powered Live2D companion that lives on your desktop with Open-LLM-VTuber, an open-source project bringing virtual interaction to life.  Check out the original repo [here](https://github.com/Open-LLM-VTuber/Open-LLM-VTuber)!

## Key Features

*   **Cross-Platform Compatibility:** Runs seamlessly on Windows, macOS, and Linux.
*   **Offline Mode:** Enjoy your AI companion without an internet connection, ensuring privacy and security.
*   **Web and Desktop Clients:** Choose your preferred mode of interaction.
*   **Advanced Interaction:**
    *   **Visual Perception:**  Your AI companion can see and react to your environment.
    *   **Voice Interruption:**  Speak and be heard without a headset.
    *   **Touch Feedback:** Interact with clicks and drags.
    *   **Live2D Expressions:** Control model expressions.
    *   **Pet Mode:** Have your companion accompany you anywhere on your screen with a transparent background.
    *   **AI's Inner Thoughts:** See expressions, thoughts, and actions without spoken words.
    *   **Proactive Speaking:** AI-initiated conversation.
    *   **Persistent Chat Logs:** Never lose your conversations.
    *   **TTS Translation:** Chat in one language, with AI speaking in another.
*   **Extensive Model Support:** Compatible with a wide range of Large Language Models (LLMs), Automatic Speech Recognition (ASR), and Text-to-Speech (TTS) engines.
*   **Highly Customizable:**
    *   **Simple Module Configuration:** Easy customization via configuration files.
    *   **Character Customization:** Import custom Live2D models and tailor your AI's persona.
    *   **Flexible Agent Implementation:** Integrate various Agent architectures.
    *   **Extensibility:** Easily add new LLMs, ASR, TTS, and more.

## Why Open-LLM-VTuber?

## Get Started

Refer to the [Quick Start Guide](https://open-llm-vtuber.github.io/docs/quick-start) in the documentation for installation and setup.

## Development and Updates

*   This project is actively developed, with frequent updates and new features planned. See the [Roadmap](https://github.com/orgs/Open-LLM-VTuber/projects/2) for details.
*   For updates, use `uv run update.py` after installation.
*   For breaking changes, please redeploy following the [latest deployment guide](https://open-llm-vtuber.github.io/docs/quick-start).

## Uninstall

Remove the project directory, including Python dependencies, models, and any extra tools like `uv`, `ffmpeg`, or `deeplx`. Review the installation guide for details.
