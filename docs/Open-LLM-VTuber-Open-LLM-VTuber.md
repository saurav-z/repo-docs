## Key Features

*   **Cross-Platform Compatibility:** Runs seamlessly on Windows, macOS, and Linux.
*   **Offline Mode:** Enjoy privacy and security with local model support, no internet needed.
*   **Interactive Clients:** Web and desktop clients offer rich interaction and customization. Desktop client supports "desktop pet mode."
*   **Advanced Interaction:**
    *   Visual perception (camera, screen recording, screenshots)
    *   Voice interruption
    *   Touch feedback
    *   Live2D expression mapping
    *   "Pet mode" (transparent background)
    *   AI's inner thought display
    *   Proactive AI speaking
    *   Chat log persistence
    *   TTS translation
*   **Extensive Model Support:** Compatible with various LLMs, ASR, and TTS engines (Ollama, OpenAI, Whisper, Coqui-TTS, etc.).
*   **Customization:**
    *   Easy module configuration.
    *   Custom Live2D models and persona.
    *   Flexible Agent implementation
    *   Modular design for easy extensibility.
