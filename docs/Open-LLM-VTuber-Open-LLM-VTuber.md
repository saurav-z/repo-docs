## Key Features

*   **Cross-Platform Compatibility:** Works seamlessly on Windows, macOS, and Linux.  Supports NVIDIA and non-NVIDIA GPUs, offering options for CPU-based processing or cloud APIs.
*   **Offline Mode:** Enjoy privacy and security with complete offline functionality using local models.
*   **Web and Desktop Clients:** Offers both a web interface and a desktop client with interactive features and customization options. The desktop client includes a transparent background pet mode.
*   **Advanced Interaction:**
    *   Visual perception via camera, screen recording, and screenshots.
    *   Microphone interruption and touch feedback.
    *   Live2D expressions for enhanced model animation.
    *   Pet mode with transparent background, global top-most, and mouse click-through.
    *   AI's inner thought display.
    *   AI proactive speaking.
    *   Persistent chat logs.
    *   TTS translation support.
*   **Extensive Model Support:**
    *   **LLMs:** Ollama, OpenAI (and compatible APIs), Gemini, Claude, Mistral, DeepSeek, Zhipu AI, GGUF, LM Studio, vLLM, etc.
    *   **ASR:** sherpa-onnx, FunASR, Faster-Whisper, Whisper.cpp, Whisper, Groq Whisper, Azure ASR, etc.
    *   **TTS:** sherpa-onnx, pyttsx3, MeloTTS, Coqui-TTS, GPTSoVITS, Bark, CosyVoice, Edge TTS, Fish Audio, Azure TTS, etc.
*   **Highly Customizable:**
    *   Easy module configuration via configuration file modifications.
    *   Character customization: Import custom Live2D models, adjust prompts, and implement voice cloning.
    *   Flexible agent implementation via Agent interface.
    *   Modular design allowing for easy expansion of features.
