## Key Features

*   **GPU & NPU Acceleration:** Optimize LLM performance using your existing hardware.
*   **Flexible Model Support:** Compatible with GGUF and ONNX models.
*   **Easy Integration:** Seamlessly integrate with your favorite OpenAI-compatible applications.
*   **Model Manager:** Download and manage models directly within the Lemonade environment.
*   **Cross-Platform:** Supports Windows and Linux operating systems.
*   **Open Source:**  Join a community of contributors.
