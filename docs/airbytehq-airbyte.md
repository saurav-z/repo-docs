## Airbyte: The Open-Source Data Integration Platform for Seamless Data Pipelines

Airbyte is an open-source data integration platform that helps you build and manage Extract, Load, and Transform (ELT) pipelines from various sources to your desired destinations, simplifying your data workflows.  [Visit the original repository](https://github.com/airbytehq/airbyte).

### Key Features

*   **Extensive Connector Catalog:** Access a vast library of 300+ pre-built connectors for APIs, databases, data warehouses, and data lakes, with more constantly being added.
*   **Open Source & Customizable:** Benefit from an open-source solution that empowers you to customize existing connectors and build your own.
*   **No-Code & Low-Code Connector Development:** Quickly create connectors using the no-code Connector Builder or low-code CDK.
*   **Flexible Orchestration:** Integrate Airbyte with popular orchestration tools like Airflow, Prefect, Dagster, and Kestra, or use the Airbyte API.
*   **User-Friendly Interface:** Easily manage your data pipelines with an intuitive user interface.
*   **Active Community:** Engage with the Airbyte community on Slack and the Airbyte forum for support, discussions, and collaboration.

### Getting Started

*   **Deploy Airbyte Open Source:** [Deploy Airbyte Open Source](https://docs.airbyte.com/quickstart/deploy-airbyte) or set up [Airbyte Cloud](https://docs.airbyte.com/cloud/getting-started-with-airbyte-cloud) to start centralizing your data.
*   **Create Connectors:** Build connectors in minutes with our [no-code Connector Builder](https://docs.airbyte.com/connector-development/connector-builder-ui/overview) or [low-code CDK](https://docs.airbyte.com/connector-development/config-based/low-code-cdk-overview).
*   **Explore Tutorials:** Dive into popular use cases with our [tutorials](https://airbyte.com/tutorials).
*   **Orchestrate Syncs:** Manage Airbyte syncs with [Airflow](https://docs.airbyte.com/operator-guides/using-the-airflow-airbyte-operator), [Prefect](https://docs.airbyte.com/operator-guides/using-prefect-task), [Dagster](https://docs.airbyte.com/operator-guides/using-dagster-integration), [Kestra](https://docs.airbyte.com/operator-guides/using-kestra-plugin), or the [Airbyte API](https://reference.airbyte.com/reference/start).

Try it out yourself with our [demo app](https://demo.airbyte.io/), explore the [full documentation](https://docs.airbyte.com/), and learn more about [recent announcements](https://airbyte.com/blog-categories/company-updates). See our [registry](https://connectors.airbyte.com/files/generated_reports/connector_registry_report.html) for a full list of connectors already available in Airbyte or Airbyte Cloud.
