## Airbyte: The Open-Source Data Integration Platform for Modern Data Teams

Airbyte is a powerful, open-source platform that simplifies data integration, enabling you to move data from any source to any destination.  [Explore the Airbyte repository](https://github.com/airbytehq/airbyte).

**Key Features:**

*   **Extensive Connector Catalog:** Access a library of 300+ pre-built connectors for popular APIs, databases, data warehouses, and data lakes.
*   **ELT Pipelines:** Build Extract, Load, and Transform (ELT) pipelines for efficient data movement.
*   **Open-Source and Customizable:** Benefit from an open-source solution that allows you to customize connectors and adapt to your specific needs.
*   **No-Code Connector Builder:** Quickly create new connectors with our user-friendly no-code Connector Builder.
*   **Orchestration Options:** Seamlessly integrate with popular orchestration tools like Airflow, Prefect, Dagster, and Kestra, as well as the Airbyte API.

## Getting Started

*   **Deploy Airbyte:** [Deploy Airbyte Open Source](https://docs.airbyte.com/quickstart/deploy-airbyte) or set up [Airbyte Cloud](https://docs.airbyte.com/cloud/getting-started-with-airbyte-cloud) to start centralizing your data.
*   **Create Connectors:** Create connectors in minutes with our [no-code Connector Builder](https://docs.airbyte.com/connector-development/connector-builder-ui/overview) or [low-code CDK](https://docs.airbyte.com/connector-development/config-based/low-code-cdk-overview).
*   **Explore Tutorials:** Discover popular use cases in our [tutorials](https://airbyte.com/tutorials).
*   **Orchestrate Syncs:** Orchestrate Airbyte syncs with [Airflow](https://docs.airbyte.com/operator-guides/using-the-airflow-airbyte-operator), [Prefect](https://docs.airbyte.com/operator-guides/using-prefect-task), [Dagster](https://docs.airbyte.com/operator-guides/using-dagster-integration), [Kestra](https://docs.airbyte.com/operator-guides/using-kestra-plugin), or the [Airbyte API](https://reference.airbyte.com/reference/start).
