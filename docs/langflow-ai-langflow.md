## Langflow: Build AI Workflows Visually and Deploy with Ease

Langflow is a powerful, open-source tool that lets you visually build, test, and deploy AI-powered applications with ease.  Visit the [Langflow GitHub repository](https://github.com/langflow-ai/langflow) to learn more.

### Key Features

*   **Visual Workflow Builder:** Design AI workflows with an intuitive drag-and-drop interface.
*   **Source Code Access:** Customize components using Python code for maximum flexibility.
*   **Interactive Playground:** Test and refine your flows in real-time with step-by-step control.
*   **Multi-Agent Orchestration:** Manage and coordinate multiple AI agents with ease, including conversation management and retrieval capabilities.
*   **API Deployment:** Deploy your workflows as APIs for easy integration with other applications.
*   **MCP Server Deployment:** Turn workflows into tools for Model Control Program (MCP) clients.
*   **Observability:** Integrate with monitoring tools like LangSmith and LangFuse for in-depth insights.
*   **Enterprise-Ready:** Benefit from robust security and scalability for production environments.

### Quickstart

1.  **Prerequisites:** Ensure you have [Python 3.10 to 3.13](https://www.python.org/downloads/release/python-3100/) and [uv](https://docs.astral.sh/uv/getting-started/installation/) installed.

2.  **Installation:** Install Langflow using pip:

    ```shell
    uv pip install langflow -U
    ```

3.  **Run Langflow:** Start the application with:

    ```shell
    uv run langflow run
    ```

4.  **Access:** Open Langflow in your browser at `http://127.0.0.1:7860`.

For more comprehensive installation options, including Docker and desktop installations, please consult the [Install Langflow](https://docs.langflow.org/get-started-installation) documentation.

### Deployment

Langflow is designed for easy deployment to various cloud platforms and supports Docker.  For detailed instructions on deploying Langflow using Docker, refer to the [Docker deployment guide](https://docs.langflow.org/deployment-docker).
