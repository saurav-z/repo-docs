## Langflow: Build AI Workflows Visually with Ease

Langflow empowers developers to build and deploy AI-powered applications and workflows using a visual, drag-and-drop interface. ([See the original repo](https://github.com/langflow-ai/langflow))

> [!CAUTION]
> - Users must update to Langflow >= 1.3 to protect against [CVE-2025-3248](https://nvd.nist.gov/vuln/detail/CVE-2025-3248)
> - Users must update to Langflow >= 1.5.1 to protect against [CVE-2025-57760](https://github.com/langflow-ai/langflow/security/advisories/GHSA-4gv9-mp8m-592r)
>
> For security information, see our [Security Policy](./SECURITY.md) and [Security Advisories](https://github.com/langflow-ai/langflow/security/advisories).

### Key Features

*   **Visual Workflow Builder:** Design AI flows quickly with an intuitive, drag-and-drop interface.
*   **Source Code Customization:** Access and modify any component using Python for advanced control.
*   **Interactive Playground:** Test and refine your flows instantly with step-by-step control.
*   **Multi-Agent Orchestration:** Manage conversations and retrieve information effectively.
*   **API Deployment:** Deploy your flows as APIs or export them as JSON for Python applications.
*   **MCP Server Deployment:** Convert your flows into tools for MCP clients.
*   **Observability:** Integrate with LangSmith, LangFuse, and other monitoring tools.
*   **Enterprise-Ready:** Benefit from built-in security and scalability features.

### Getting Started

Langflow requires [Python 3.10 to 3.13](https://www.python.org/downloads/release/python-3100/) and [uv](https://docs.astral.sh/uv/getting-started/installation/).

1.  **Install Langflow:**

    ```shell
    uv pip install langflow -U
    ```

2.  **Run Langflow:**

    ```shell
    uv run langflow run
    ```

3.  **Access Langflow:** Open your web browser and go to `http://127.0.0.1:7860`.

For comprehensive installation instructions, including Docker and desktop options, please consult the [Install Langflow](https://docs.langflow.org/get-started-installation) documentation.

### Deployment Options
