# Model Context Protocol Servers: Unleash the Power of AI with Secure Tool Access

**Give your Large Language Models (LLMs) secure and controlled access to tools and data sources with the Model Context Protocol (MCP) ‚Äì the open standard for AI tool integrations.** This repository ([original repo](https://github.com/modelcontextprotocol/servers)) provides reference implementations, community-built servers, and resources to help you leverage MCP.

## Key Features

*   **Secure Tool Access:** Enable LLMs to interact safely with various tools and data sources.
*   **Extensibility:** Build and integrate custom servers to extend LLM capabilities.
*   **Community-Driven:** Benefit from a growing ecosystem of community-built servers and resources.
*   **SDK Support:** Utilize official SDKs for C#, Java, Kotlin, Python, and Typescript for easier development.
*   **Versatile Implementations:** Explore diverse use cases through a wide array of reference and community servers.

## Reference Servers

These servers showcase MCP features and are implemented using the official SDKs:

*   **Everything:** Comprehensive test server with prompts, resources, and tools.
*   **Fetch:** Web content fetching and conversion for efficient LLM use.
*   **Filesystem:** Secure file operations with customizable access controls.
*   **Git:** Tools for reading, searching, and manipulating Git repositories.
*   **Memory:** Knowledge graph-based persistent memory system.
*   **Sequential Thinking:** Dynamic and reflective problem-solving through thought sequences.
*   **Time:** Time and timezone conversion capabilities.

## üöÄ Getting Started

1.  **Installation (Typescript Servers):**

    ```bash
    npx -y @modelcontextprotocol/server-memory  # Example
    ```

2.  **Installation (Python Servers):**

    ```bash
    uvx mcp-server-git # Example (recommended)
    # OR
    pip install mcp-server-git
    python -m mcp_server_git
    ```

3.  **Client Configuration (e.g., Claude Desktop):**

    ```json
    {
      "mcpServers": {
        "memory": {
          "command": "npx",
          "args": ["-y", "@modelcontextprotocol/server-memory"]
        }
      }
    }
    ```
