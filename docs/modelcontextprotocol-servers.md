# Model Context Protocol (MCP) Servers: Empowering LLMs with Secure Tool and Data Access

**Unlock the potential of Large Language Models (LLMs) by granting them secure and controlled access to tools and data using Model Context Protocol (MCP).** Explore a rich ecosystem of reference implementations and community-built servers, enhancing LLMs' capabilities for various applications.  Explore the original repo: [https://github.com/modelcontextprotocol/servers](https://github.com/modelcontextprotocol/servers)

## Key Features

*   **Reference Implementations:** Explore examples of MCP servers in various languages ([C#](https://github.com/modelcontextprotocol/csharp-sdk), [Java](https://github.com/modelcontextprotocol/java-sdk), [Kotlin](https://github.com/modelcontextprotocol/kotlin-sdk), [Python](https://github.com/modelcontextprotocol/python-sdk), [Typescript](https://github.com/modelcontextprotocol/typescript-sdk)).
*   **Extensible Ecosystem:** Discover a growing collection of servers for diverse functionalities.
*   **Secure Access:** Control LLM access to tools and data sources, ensuring safe and responsible use.
*   **Community-Driven Development:** Contribute your own servers and enhancements, helping to expand the MCP ecosystem.

## Reference Servers

These servers showcase core MCP features and utilize official MCP SDKs:

*   **Everything**:  A comprehensive reference and testing server, providing prompts, resources, and tools.
*   **Fetch**: Retrieves and converts web content to enhance LLM utilization.
*   **Filesystem**: Implements secure file operations with configurable access controls.
*   **Git**: Enables reading, searching, and manipulating Git repositories.
*   **Memory**:  A knowledge graph-based persistent memory system.
*   **Sequential Thinking**: Facilitates dynamic and reflective problem-solving through thought sequences.
*   **Time**: Offers time and timezone conversion capabilities.

### Archived Servers

These reference servers have been archived and can be found in the [servers-archived](https://github.com/modelcontextprotocol/servers-archived) repository.
