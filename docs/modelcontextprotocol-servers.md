# Model Context Protocol (MCP) Servers: Unlock AI's Potential with Secure Tool Access

**Harness the power of AI by giving Large Language Models (LLMs) secure and controlled access to tools and data sources with Model Context Protocol servers.** ([View Original Repo](https://github.com/modelcontextprotocol/servers))

## Key Features:

*   **Reference Implementations:** Explore a diverse collection of reference servers showcasing MCP's versatility.
*   **Extensible SDKs:** Utilize the official MCP SDKs for C#, Go, Java, Kotlin, Python, and TypeScript to build your own servers.
*   **Official Integrations:** Benefit from production-ready MCP servers maintained by leading companies.
*   **Community-Built Servers:** Discover a rapidly growing ecosystem of community-developed servers spanning diverse domains.
*   **Secure and Controlled Access:** Enable LLMs to interact with tools and data while maintaining security and control.

## Reference Servers:

Explore these reference servers to learn how to implement MCP features and use the official SDKs.

*   **Everything:** A comprehensive test server with prompts, resources, and tools.
*   **Fetch:** Fetch and convert web content for efficient LLM usage.
*   **Filesystem:** Secure file operations with configurable access controls.
*   **Git:** Read, search, and manipulate Git repositories.
*   **Memory:** A persistent, knowledge graph-based memory system.
*   **Sequential Thinking:** Dynamic and reflective problem-solving through thought sequences.
*   **Time:** Time and timezone conversion capabilities.
