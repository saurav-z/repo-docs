## Unleash the Power of AI Translation with AiNiee: Your One-Stop Solution for Seamless Text Conversion

AiNiee is a versatile AI-powered translation tool designed to effortlessly translate complex text, including games, books, subtitles, and documents, with unmatched speed and accuracy.  [**Explore the AiNiee Repo**](https://github.com/NEKOparapa/AiNiee) for more details!

### Key Features:

*   **Comprehensive Format Support:**
    *   üéÆ **Game Translation:** Deeply supports tools like Mtool, Renpy, Translator++, ParaTranzr, VNText, and SExtractor for game text extraction and translation.
    *   üìö **Wide Compatibility:** Processes I18Next data, Epub/TXT e-books, Srt/Vtt/Lrc subtitles, Word/PDF/MD documents, and more.

*   **Intelligent & Efficient:**
    *   üöÄ **One-Click Operation:** Simply drag and drop, and AiNiee automatically detects files and languages, simplifying the entire process.
    *   ‚è±Ô∏è **Lightning-Fast Translation:** Get your translations done in a matter of minutes, saving you valuable time.

*   **Enhanced Quality for Long Texts:**
    *   üéØ **Overcoming Limitations:** Employs advanced techniques such as lightweight translation formats, chain-of-thought translation, AI terminology tables, and context association to ensure coherent and accurate long-text translations.
    *   üíé **Quality Enhancement:** Supports prompt adjustments for basic prompts, character introductions, background settings, and translation styles. Features one-click AI refinement, one-click AI formatting, and one-click term extraction to meet the needs of users with higher translation quality requirements.

### Other Instructions

</summary>

*   `Multiple Key Polling`
    > If you want to use multiple keys to share the consumption pressure and speed up the translation according to the number of keys, please use the keys of the same type of account, and add a comma in English between each key, and do not line break. For example: key1,key2,key3

*   `Batch File Translation`
    > Just put all the files that need to be translated in the input folder, and it also supports multi-folder structure

*   `Configuration Migration`
    > The configuration information will be stored in config.json in resource. Download the new version and copy it to the new version of resource.

## Contribution Guide[![](https://raw.githubusercontent.com/aregtech/areg-sdk/master/docs/img/pin.svg)](#contribution-guide)

</summary>

*   **`Develop Enhanced Plugins`**: Please refer to the [Plugin Writing Guide](https://github.com/NEKOparapa/AiNiee/blob/main/PluginScripts/README.md) to develop more powerful function plugins

*   **`Improve or Add Support for Files`**: Requires some coding ability. Pull the source code for improvements. The file reading code is in the ModuleFolders\\FileReader and FileOutputer folders. [Reader and Writer System Writing Guide](https://github.com/NEKOparapa/AiNiee/blob/main/ModuleFolders/FileAccessor/README.md). UI support in UserInterface\\Setting's ProjectSettingsPage.

*   **`Improve Regex Library`**: The completeness of the regular expression library will greatly help the embedded work in the game, and benefit the next game translation work and benefit other translation users. The regular expression library is in the [Resource\\Regex](https://github.com/NEKOparapa/AiNiee/blob/main/Resource/Regex/regex.json) folder

*   **`Improve Interface Translation`**: The UI text of multi-language interfaces may not be translated accurately and appropriately. You can submit your modification opinions or modify them directly. The localization text is in the [Resource\\Localization](https://github.com/NEKOparapa/AiNiee/tree/main/Resource/Localization) folder
