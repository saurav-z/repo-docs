## üöÄ Supercharge Your Translations with AiNiee: The AI Translation Powerhouse!

AiNiee is a cutting-edge AI-powered translation tool designed to effortlessly translate complex text from games, books, subtitles, and more.  [Check it out on GitHub!](https://github.com/NEKOparapa/AiNiee)

---

## üîë Key Features

*   **üåê Wide Format Support:**
    *   üéÆ **Game Translation:** Compatible with Mtool, Renpy, Translator++, ParaTranzr, VNText, SExtractor, and more.
    *   üìö **Diverse Content:** Processes I18Next data, Epub/TXT ebooks, Srt/Vtt/Lrc subtitles, Word/PDF/MD documents, and more.

*   **üí° Intelligent & Efficient:**
    *   üöÄ **One-Click Operation:** Automatically identifies files and languages; just drag and drop.
    *   ‚è±Ô∏è **Blazing Fast:** Get your translations in minutes!

*   **‚ú® Optimized for Quality:**
    *   üéØ **Advanced Techniques:** Employs light translation formats, thought chain translation, AI terminology tables, and context association for accurate long-text translation.
    *   üíé **Quality Focused:** Offers prompts like basic tips, character introductions, background settings, and translation style adjustments, with features like one-click AI refinement, layout, and terminology extraction.

### üí° Other Notes
</summary>

*   ` Multiple Key Rotation`
    > If you want to use multiple keys to share the consumption pressure and speed up the translation process, please use keys of the same type account, and enter the keys separated by commas in English, and do not use line breaks. For example: key1,key2,key3

*   ` Batch File Translation`
    > Just put all the files that need to be translated in the input folder, and it also supports multiple folder structures

*   ` Configuration Migration`
    > Configuration information is stored in resource's config.json. Download a new version can copy it to the new version's resource directory.

## ü§ù Contributing Guide
</summary>

*   **`Develop Enhancement Plugins`**:  Follow the [plugin writing guide](https://github.com/NEKOparapa/AiNiee/blob/main/PluginScripts/README.md) to create powerful plugins.

[Reader/Writer System Guide](https://github.com/NEKOparapa/AiNiee/blob/main/ModuleFolders/FileAccessor/README.md). UI support in `UserInterface\Setting` within the `ProjectSettingsPage`.

*   **`Improve Regex Library`**:  Completing the Regex library will greatly aid in game embedding and benefit future translation projects. The Regex library is in the `Resource\Regex` folder.

*   **`Improve Interface Translation`**:  If the multi-language UI text isn't accurate, suggest changes, or modify directly. Localization text is in the `Resource\Localization` folder.
