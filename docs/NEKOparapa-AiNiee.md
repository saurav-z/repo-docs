## üöÄ Unlock the Power of AI Translation with AiNiee!

**AiNiee is your all-in-one AI translation tool, designed to effortlessly translate complex text from games, books, subtitles, and documents.**

*   **Comprehensive Format Support:**
    *   üéÆ **Game Translation:** Deeply supports game text extraction tools like Mtool, Renpy, Translator++, ParaTranzr, VNText, and SExtractor.
    *   üìö **Versatile Compatibility:** Processes I18Next data, Epub/TXT e-books, Srt/Vtt/Lrc subtitles, Word/PDF/MD documents, and more.

*   **Intelligent & Efficient:**
    *   üöÄ **One-Click Operation:** Simply drag and drop, and AiNiee automatically detects files and languages, eliminating the need for manual setup.
    *   ‚è±Ô∏è **Lightning-Fast Translation:** Get translated text quickly, saving you valuable time.

*   **Enhanced Quality for Long Texts:**
    *   üéØ **Overcome Limitations:** Uses lightweight translation formats, chain-of-thought translation, AI terminology tables, and context association to ensure fluency and accuracy.
    *   üíé **Quality Enhancement:** Supports prompts for basic instructions, character introductions, background settings, and translation styles, with features for AI refinement, formatting, and terminology extraction to meet high-quality translation requirements.

## üõ†Ô∏è Getting Started with AiNiee: Three Simple Steps

1.  **Configure Your API Interface:**

    *   **Online Interfaces:** (Paid, high cost-effectiveness, no GPU required, supports all languages)
        *   [DeepSeek Interface Setup](https://github.com/NEKOparapa/AiNiee/wiki/QuickStartDeepSeek)
    *   **Online Interfaces:** (Alternative if DeepSeek is unavailable)
        *   [HuoShan Engine Interface Setup](https://github.com/NEKOparapa/AiNiee/wiki/QuickStartHuo)
    
    >   <img src="https://raw.githubusercontent.com/NEKOparapa/AiNiee/main/Example%20image/%E4%B8%89%E6%AD%A5%E8%B5%B0/%E7%AC%AC%E4%B8%80%E6%AD%A5.png" alt="Step 1: Configure Interface" width="400">

2.  **Drag and Drop Your Files:**

    *   Place the original files in a new folder and drag the folder into the designated area. Novels, subtitles, and documents can be translated directly. Games require text extraction tools.

    >   <img src="https://raw.githubusercontent.com/NEKOparapa/AiNiee/main/Example%20image/%E4%B8%89%E6%AD%A5%E8%B5%B0/%E7%AC%AC%E4%BA%8C%E6%AD%A5.png" alt="Step 2: Drag and Drop Files" width="400">

3.  **Start Translating!**

    *   Click the "Start" button and wait for the task to complete.

    >   <img src="https://raw.githubusercontent.com/NEKOparapa/AiNiee/main/Example%20image/%E4%B8%89%E6%AD%A5%E8%B5%B0/%E7%AC%AC%E4%B8%89%E6%AD%A5.png" alt="Step 3: Start Translation" width="400">

### üí° Other Notes
</summary>

*   `Multi-Key Polling`:
    >   To use multiple keys for distributed processing and speed up translation (based on the number of keys), use keys of the same account type, and separate them with commas without line breaks (e.g., key1,key2,key3).
*   `Batch File Translation`:
    >   Place all files to be translated in the input folder, also supports multi-folder structure.
*   `Configuration Migration`:
    >   Configuration information is stored in `resource/config.json`. Copy this file to the new version's `resource` folder to migrate settings.

## ü§ù Contributing
</summary>

*   **`Develop Enhanced Plugins`**: Follow the [Plugin Writing Guide](https://github.com/NEKOparapa/AiNiee/blob/main/PluginScripts/README.md) to create more powerful plugins.

*   **`Improve or Add Support for Files`**: Requires some programming skills; pull the source code for improvements. The file reading code is in the `ModuleFolders\FileReader` and `FileOutputer` folders. See the [Writer System Guide](https://github.com/NEKOparapa/AiNiee/blob/main/ModuleFolders/FileAccessor/README.md). UI support in `UserInterface\Setting` and  `ProjectSettingsPage`.

*   **`Improve the Regular Expression Library`**: Completeness of the regex library will greatly help in-game embedding work, benefiting future game translation efforts and other translators. The regex library is in the `Resource\Regex` folder.

*   **`Improve the UI Translation`**: The UI text in multiple languages might not be accurately or appropriately translated. You can submit your suggestions or modify the text directly. Localized text is in the `Resource\Localization` folder.
