## AiNiee: The Ultimate AI Translation Tool for Games, Books, and More

**Tired of struggling with complex text translation?** AiNiee streamlines the process, offering effortless AI translation for games, books, subtitles, and documents.

*   **Versatile Format Support:**
    *   üéÆ **Game Translation:** Deep support for popular game text extraction tools like Mtool, Renpy, Translator++, ParaTranzr, VNText, and SExtractor.
    *   üìö **Wide Range of Formats:**  Easily handles I18Next data, Epub/TXT e-books, Srt/Vtt/Lrc subtitles, and Word/PDF/MD documents.

*   **Intelligent & Efficient Translation:**
    *   üöÄ **One-Click Operation:** Simply drag, drop, and AiNiee automatically recognizes files and languages, eliminating the need for tedious settings.
    *   ‚è±Ô∏è **Blazing-Fast Translations:** Get your translated text in the time it takes to enjoy a soda!

*   **Optimized for Long Texts & High Quality:**
    *   üéØ **Advanced Techniques:** Utilizes techniques like lightweight translation formats, chain-of-thought translation, AI terminology tables, and contextual association to ensure accurate and fluent translations for long texts.
    *   üíé **Quality Enhancement:** Includes features like customizable prompts (basic hints, character introductions, background settings, translation styles), one-click AI refinement, one-click AI formatting, and one-click terminology extraction for users who demand higher translation quality.

### Other Notes

</summary>

*   **Multiple Key Rotation:**

    *   If you need to use multiple keys to share the translation load, use keys of the same type. Separate keys with commas (no line breaks). Example: `key1,key2,key3`
*   **Batch File Translation:**

    *   Place all files to be translated in the input folder. Supports multi-folder structures.
*   **Configuration Migration:**

    *   Configuration information is stored in `resource/config.json`. Copy this file to the new version's `resource` folder after downloading an update.

## Contribution Guide [<img src="https://raw.githubusercontent.com/aregtech/areg-sdk/master/docs/img/pin.svg">](#Ë¥°ÁåÆÊåáÂçó)

</summary>

*   **`Develop Enhanced Plugins`**:  Follow the [Plugin Writing Guide](https://github.com/NEKOparapa/AiNiee/blob/main/PluginScripts/README.md) to create more powerful plugins.

*   **`Improve or Add File Support`**: Requires some coding ability. Make improvements by pulling the source code.  File reading/writing code is in the `ModuleFolders\FileReader` and `FileOutputer` folders. [Writing Guide for Readers and Writers](https://github.com/NEKOparapa/AiNiee/blob/main/ModuleFolders/FileAccessor/README.md). UI support is in `UserInterface\Setting` > `ProjectSettingsPage`.

*   **`Improve Regular Expression Library`**: A complete regex library will greatly help with game embedding and benefit future translation work. The regex library is in the `Resource\Regex` folder [Resource\Regex](https://github.com/NEKOparapa/AiNiee/blob/main/Resource/Regex/regex.json)

*   **`Improve UI Translation`**: If the multilingual UI text translation is not accurate, submit your suggestions or modify the text directly. Localized text is in the `Resource\Localization` folder. [Resource\Localization](https://github.com/NEKOparapa/AiNiee/tree/main/Resource/Localization)
