# AiNiee: Effortless AI Translation for Games, Books, and More üöÄ

AiNiee is your all-in-one solution for AI-powered translation, simplifying the process of translating complex text from games, books, subtitles, and documents with ease.  **[Check out the original repository here!](https://github.com/NEKOparapa/AiNiee)**

---

## Key Features üîë

*   **Comprehensive Format Support**:
    *   üéÆ **Game Translation**: Seamlessly translates text from Mtool, Renpy, Translator++, ParaTranzr, VNText, SExtractor, and more.
    *   üìö **Wide Compatibility**: Handles I18Next data, Epub/TXT e-books, Srt/Vtt/Lrc subtitles, Word/PDF/MD documents, and various other formats.

*   **Smart & Efficient Workflow**:
    *   üöÄ **One-Click Operation**: Automatically detects files and languages, requiring no complex setup.
    *   ‚è±Ô∏è **Blazing Fast**: Get translations completed in minutes.

*   **Enhanced Quality for Long Texts**:
    *   üéØ **Overcoming Limitations**: Utilizes light translation formats, chain-of-thought translation, AI terminology tables, and context association for accurate and coherent long-text translations.
    *   üíé **Quality Control**: Supports prompt adjustments (basic prompts, character introductions, background settings, translation styles) and offers one-click AI refinement, formatting, and terminology extraction for users seeking high-quality translations.

*   `Multi-key Rotation`
    >   To use multiple keys for faster translation (based on the number of keys), please use keys of the same type and separate them with commas without line breaks. For example: key1,key2,key3

*   `Batch File Translation`
    >   Simply place all files to be translated in the input folder. Multi-folder structures are also supported.

*   `Configuration Migration`
    >   Configuration information is stored in resource's config.json. Copy it to the new version's resource folder when updating.
