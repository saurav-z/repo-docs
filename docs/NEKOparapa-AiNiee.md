## üöÄ Supercharge Your Translations with AiNiee: The AI-Powered Translation Tool

AiNiee simplifies complex text translation, providing one-click AI translation for games, books, subtitles, and documents.  **[Click here to explore the AiNiee project on GitHub!](https://github.com/NEKOparapa/AiNiee)**

---

## Key Features:

*   **üåê Comprehensive Format Support:**
    *   üéÆ **Game Translation:** Fully supports game text extraction tools like Mtool, Renpy, Translator++, ParaTranzr, VNText, and SExtractor.
    *   üìö **Diverse File Compatibility:** Handles a wide range of formats including I18Next data, Epub/TXT ebooks, Srt/Vtt/Lrc subtitles, and Word/PDF/MD documents.

*   **üí° Smart & Efficient Workflow:**
    *   üöÄ **One-Click Operation:**  Automatically detects file types and languages with a simple drag-and-drop interface, eliminating the need for complex settings.
    *   ‚è±Ô∏è **Lightning-Fast Translation:** Get your translations in minutes, saving you valuable time.

*   **‚ú® Enhanced Quality for Long Texts:**
    *   üéØ **Advanced Techniques:** Employs techniques like lightweight translation formats, chain-of-thought translation, AI terminology, and contextual understanding to ensure accuracy and fluency.
    *   üíé **Quality Customization:** Offers features for refining translations, including customizable prompts for base hints, character introductions, background settings, and translation styles, plus one-click AI polishing, formatting, and terminology extraction.

### üìù Other Notes

*   `Multi-Key Rotation`
    >   To use multiple keys for faster translation and distribute consumption, use keys of the same type, separated by commas (no line breaks). Example: `key1,key2,key3`

*   `Batch File Translation`
    >   Place all files for translation into the input folder. Multi-folder structures are also supported.

*   `Configuration Migration`
    >   Configuration information is stored in `resource/config.json`.  Copy this file to the `resource` folder of the new version.

## ü§ù Contributing [![](https://raw.githubusercontent.com/aregtech/areg-sdk/master/docs/img/pin.svg)](#contributing)
</summary>

*   **`Develop Enhanced Plugins`**: Follow the [Plugin Development Guide](https://github.com/NEKOparapa/AiNiee/blob/main/PluginScripts/README.md) to create more powerful plugins.
*   **`Improve or Add File Support`**: Requires some coding skills.  Improve the code by pulling the source code.  File reading and writing code can be found in the `ModuleFolders/FileReader` and `FileOutputer` folders.  Refer to the [Reader/Writer System Guide](https://github.com/NEKOparapa/AiNiee/blob/main/ModuleFolders/FileAccessor/README.md). UI support can be found in `UserInterface\Setting` in `ProjectSettingsPage`.
*   **`Improve Regular Expression Library`**: A complete regular expression library is crucial for game embedding.  Improve the regular expression library in the [Resource\Regex](https://github.com/NEKOparapa/AiNiee/blob/main/Resource/Regex/regex.json) folder.
*   **`Improve Interface Translation`**: If the multi-language UI text needs improvement, submit your suggestions or modify directly.  Localization texts are in the [Resource\Localization](https://github.com/NEKOparapa/AiNiee/tree/main/Resource/Localization) folder.
