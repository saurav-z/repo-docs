## Effortlessly Translate Complex Text with AiNiee: Your AI-Powered Translation Solution

AiNiee is your all-in-one AI translation tool, designed to streamline the translation of games, books, subtitles, and documents with ease.  Check out the [original repository](https://github.com/NEKOparapa/AiNiee) for more information and to get started!

### Key Features:

*   üéÆ **Versatile Format Support:**  Seamlessly translate a wide range of file types, including those from Mtool, Renpy, Translator++, ParaTranzr, VNText, SExtractor and many more for game translation, plus I18Next data, Epub/TXT ebooks, Srt/Vtt/Lrc subtitles, and Word/PDF/MD documents.
*   üöÄ **Intelligent & Efficient:**  Translate with a simple drag-and-drop interface; AiNiee automatically identifies files and languages, eliminating the need for manual configuration.
*   ‚è±Ô∏è **Lightning-Fast Translation:** Get your translations in record time - just the time it takes to drink a soda!
*   üéØ **Optimized for Long Texts:**  Achieve superior quality with advanced features such as lightweight translation formats, chain-of-thought translation, AI terminology management, and contextual understanding, guaranteeing coherent and accurate translations, even for lengthy content.
*   üíé **High-Quality Output:** Fine-tune your translations with options for custom prompts (base prompts, character introductions, background settings, translation styles), AI refinement, AI layout, and terminology extraction to meet the highest quality standards.

### Other Notes
</summary>

*   `Multiple Key Rotation`
    > To utilize multiple API keys for load balancing and faster translation, use keys from the same provider, separated by commas (without line breaks).  For example: `key1,key2,key3`

*   `Batch File Translation`
    >  Place all files to be translated in the input folder; supports multi-folder structures.

*   `Configuration Migration`
    > Configuration information is stored in `resource/config.json`. Copy this file to the new version's resource folder when updating.

## Contribution Guide[![](https://raw.githubusercontent.com/aregtech/areg-sdk/master/docs/img/pin.svg)](#Ë¥°ÁåÆÊåáÂçó)
</summary>

*   **`Develop Enhancement Plugins`**: Follow the [plugin writing guide](https://github.com/NEKOparapa/AiNiee/blob/main/PluginScripts/README.md) to create more powerful plugins.
*   **`Improve or Add File Support`**: Requires some coding skills; pull the source code and make improvements.  File reading/writing code is in the ModuleFolders\FileReader and FileOutputer folders.  See the [reader/writer system writing guide](https://github.com/NEKOparapa/AiNiee/blob/main/ModuleFolders/FileAccessor/README.md). UI support is in UserInterface\Setting\ProjectSettingsPage.
*   **`Improve Regular Expression Library`**:  A complete regular expression library is crucial for game integration.  The regex library is in the [Resource\Regex](https://github.com/NEKOparapa/AiNiee/blob/main/Resource/Regex/regex.json) folder.
*   **`Improve UI Translation`**: You can submit suggestions or directly modify the UI translation for more accurate and appropriate multi-language text.  Localization texts are in the [Resource\Localization](https://github.com/NEKOparapa/AiNiee/tree/main/Resource/Localization) folder.
