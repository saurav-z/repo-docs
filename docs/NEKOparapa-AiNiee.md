## Revolutionize Your Translations with AiNiee: The AI-Powered Translation Tool

**AiNiee** is a powerful and versatile AI translation tool designed to streamline the translation of complex text from games, books, subtitles, and documents.  Access the original repository here:  [AiNiee on GitHub](https://github.com/NEKOparapa/AiNiee).

---

## Key Features

*   **Versatile Format Support:** AiNiee seamlessly handles a wide array of formats, ensuring compatibility with your diverse translation needs.
    *   üéÆ **Game Translation:** Deeply supports game text extraction tools like Mtool, Renpy, Translator++, and more.
    *   üìö **Extensive File Support:**  Effortlessly processes I18Next data, Epub/TXT ebooks, Srt/Vtt/Lrc subtitles, Word/PDF/MD documents, and more.

*   **Smart & Efficient Workflow:**  Get high-quality translations quickly and easily with AiNiee's automated processes.
    *   üöÄ **One-Click Operation:**  Simply drag and drop, and AiNiee automatically identifies files and languages, requiring no manual configuration.
    *   ‚è±Ô∏è **Lightning-Fast Translations:**  Get translated content quickly, saving you valuable time.

*   **Optimized for Long Texts & High Quality:**  Achieve superior translation quality for even the most complex documents.
    *   üéØ **Advanced Techniques:** Utilizes advanced techniques like lightweight translation formatting, thought chain translation, AI terminology management, and contextual association to ensure coherence and accuracy in long text translations.
    *   üíé **Quality Enhancement:** Offers features like AI polishing, AI formatting, and terminology extraction, catering to users who demand the highest quality translations. Includes support for custom prompts such as basic hints, character introductions, and background settings.

*   `Multiple Key Rotation`
    >   If you want to use multiple keys to share the consumption pressure and speed up translation, please use keys of the same type, and add an English comma between each key, without line breaks. For example: key1,key2,key3

*   `Batch File Translation`
    >   Just put all the files you need to translate in the input folder, and it also supports multi-folder structures

*   `Configuration Migration`
    >   Configuration information will be stored in resource's config.json, you can copy it to the resource of the new version after downloading it.

*   **`Improve Regex Library`**: The completeness of the Regex library will greatly help the in-game embedding work, and will benefit the next game translation work and benefit other translation users. *   **`Improve Interface Translation`**: The UI text of the multi-language interface may not be translated accurately and appropriately, you can submit your modification suggestions, or modify it directly.
