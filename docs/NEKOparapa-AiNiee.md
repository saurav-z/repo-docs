## AiNiee: Your All-in-One AI Translation Tool 🚀

**AiNiee is your go-to solution for effortlessly translating complex text, including games, books, subtitles, and documents, with AI-powered precision.**

*   **Key Features:**
    *   🎮 **Game Translation Mastery:**  Deep support for popular game text extraction tools (Mtool, Renpy, Translator++, ParaTranzr, VNText, SExtractor, and more).
    *   📚 **Versatile Format Compatibility:** Handles a wide range of formats, including I18Next data, Epub/TXT ebooks, Srt/Vtt/Lrc subtitles, Word/PDF/MD documents, and more.
    *   🚀 **One-Click Efficiency:**  Automatic file and language detection – just drag and drop!
    *   ⏱️ **Lightning-Fast Translations:** Get your translations done quickly, saving you valuable time.
    *   🎯 **Advanced Long-Text Optimization:** Leverages techniques like light translation formats, thought chain translation, AI terminology tables, and contextual association for accurate and coherent translations of lengthy content.
    *   💎 **High-Quality Refinement:** Supports prompt customization (basic prompts, character introductions, background settings, translation styles), one-click AI refinement, one-click AI typesetting, and one-click terminology extraction for superior translation quality.

## Getting Started with AiNiee: A Simple 3-Step Process

1.  **Configure Your API:**
    >   <img src="https://raw.githubusercontent.com/NEKOparapa/AiNiee/main/Example%20image/%E4%B8%89%E6%AD%A5%E8%B5%B0/%E7%AC%AC%E4%B8%80%E6%AD%A5.png">

    *   **Online APIs:** Cost-effective, no GPU required, and support all languages.
        *   [DeepSeek API Setup Guide](https://github.com/NEKOparapa/AiNiee/wiki/QuickStartDeepSeek)
        *   [HuoShan (Volcano Engine) API Setup Guide (Alternative if DeepSeek is unavailable)](https://github.com/NEKOparapa/AiNiee/wiki/QuickStartHuo)

2.  **Drag and Drop Your Files:**
    >   <img src="https://raw.githubusercontent.com/NEKOparapa/AiNiee/main/Example%20image/%E4%B8%89%E6%AD%A5%E8%B5%B0/%E7%AC%AC%E4%BA%8C%E6%AD%A5.png">

    *   Place your source files in a dedicated folder and drag the folder into the designated area. Novels, subtitles, and documents can be translated directly. Games require text extraction tools.

3.  **Start Translating!**

    >   <img src="https://raw.githubusercontent.com/NEKOparapa/AiNiee/main/Example%20image/%E4%B8%89%E6%AD%A5%E8%B5%B0/%E7%AC%AC%E4%B8%89%E6%AD%A5.png">

    *   Click the "Start" button and let AiNiee handle the rest.

    *   [Download AiNiee](https://github.com/NEKOparapa/AiNiee/releases)

### Other Notes
</summary>

*   `Multi-Key Rotation`
    >   To accelerate translation using multiple keys to share the load, use keys from the same account type, separating each key with a comma without line breaks. Example: key1,key2,key3

*   `Batch File Translation`
    >   Place all files to be translated in the input folder; multi-folder structures are also supported.

*   `Configuration Migration`
    >   Configuration information is stored in `resource/config.json`. Copy this file to the `resource` folder of the new version.

## Contributing

<details>
<summary>
### Contribution Guide
</summary>

*   **`Develop Enhanced Plugins`**:  Follow the [Plugin Writing Guide](https://github.com/NEKOparapa/AiNiee/blob/main/PluginScripts/README.md) to create more powerful plugins.
*   **`Improve or Add Support for Files`**: Requires some coding ability; pull the source code for improvements.  File reading code is in the `ModuleFolders/FileReader` and `FileOutputer` folders.  See the [Reader/Writer System Writing Guide](https://github.com/NEKOparapa/AiNiee/blob/main/ModuleFolders/FileAccessor/README.md). UI support is in the `UserInterface\Setting`'s `ProjectSettingsPage`.
*   **`Improve the Regular Expression Library`**:  Completing the regex library will greatly help with game embedding and benefit future translation work and other translators.  The regex library is in the [Resource\Regex](https://github.com/NEKOparapa/AiNiee/blob/main/Resource/Regex/regex.json) folder.
*   **`Improve Interface Translations`**:  UI text in multiple languages may not be accurately translated.  Submit your suggestions or directly modify the text.  Localization texts are in the [Resource\Localization](https://github.com/NEKOparapa/AiNiee/tree/main/Resource/Localization) folder.

</details>
