## AiNiee: Effortless AI Translation for Games, Books, and More!

**AiNiee** simplifies AI translation, making complex text translation a breeze.  

### Key Features:

*   **Comprehensive Format Support**:
    *   üéÆ **Game Translation**:  Seamlessly supports game text extraction tools like Mtool, Renpy, Translator++, ParaTranzr, VNText, and SExtractor.
    *   üìö **Wide Variety of Formats**: Processes I18Next data, Epub/TXT ebooks, Srt/Vtt/Lrc subtitles, Word/PDF/MD documents, and more.

*   **Smart and Efficient**:
    *   üöÄ **One-Click Operation**: Drag, drop, and translate!  Automatic file and language detection; no setup required.
    *   ‚è±Ô∏è **Blazing Fast**: Get your translations in minutes.

*   **Optimized for Quality**:
    *   üéØ **Enhanced Accuracy**:  Utilizes lightweight translation formats, chain-of-thought translation, AI terminology tables, and context-aware technology for accurate and coherent translations, even for long texts.
    *   üíé **Advanced Quality Control**:  Supports prompt adjustments for basic hints, character introductions, background settings, and translation style. Includes one-click AI refinement, one-click AI formatting, and one-click term extraction for high-quality results.

### Additional Information
</summary>

*   **Multiple API Keys**:  To share the workload using several keys and accelerate the translation process, enter the API keys of the same type accounts with English commas as separators (no line breaks). For example: `key1,key2,key3`.
*   **Batch File Translation**:  Place all files to be translated in the input folder. Nested folder structures are also supported.
*   **Configuration Migration**: Configuration information is stored in `resource/config.json`. Copy this file to the same location in a newer version to migrate your settings.

## Contribution Guide [![](https://raw.githubusercontent.com/aregtech/areg-sdk/master/docs/img/pin.svg)](#contribution-guide)
</summary>

*   **`Develop Enhanced Plugins`**: Follow the [Plugin Writing Guide](https://github.com/NEKOparapa/AiNiee/blob/main/PluginScripts/README.md) to create more powerful plugins.
*   **`Improve or Add Support for File Formats`**: Requires some coding skills. Pull the source code and make improvements. File reading code is located in the `ModuleFolders/FileReader` and `FileOutputer` folders.  See the [Reader/Writer System Guide](https://github.com/NEKOparapa/AiNiee/blob/main/ModuleFolders/FileAccessor/README.md). UI support is located in `UserInterface/Setting` in the `ProjectSettingsPage`.
*   **`Improve Regular Expression Library`**: A complete regex library significantly benefits game embedding and future translation efforts. The regex library is in the `Resource/Regex/regex.json` folder.
*   **`Improve UI Translations`**: If the UI text is not translated accurately, submit your suggestions or make changes directly. Localized text is located in the `Resource/Localization` folder.
