## Effortlessly Translate Anything with AiNiee: Your AI-Powered Translation Tool

AiNiee is a powerful AI-driven translation tool designed to quickly and accurately translate complex text formats like games, books, subtitles, and documents. Visit the [original repo](https://github.com/NEKOparapa/AiNiee) for more details!

### Key Features:

*   üéÆ **Comprehensive Format Support**:  Translate a wide variety of file types including:
    *   Game files from Mtool, Renpy, Translator++, ParaTranzr, VNText, SExtractor and more.
    *   Epub/TXT eBooks, Srt/Vtt/Lrc Subtitles, Word/PDF/MD Documents, and I18Next data.
*   üöÄ **Intelligent & Efficient**:
    *   One-click operation: Drag and drop, and the AI automatically detects the file type and language.
    *   Speedy Translations: Get your translated content in a matter of minutes!
*   üéØ **Enhanced Long-Text Handling**:
    *   Advanced Techniques: Optimized for long-form content using light-weight translation formats, chain-of-thought translation, AI terminology tables, and context-aware processing to ensure coherent and accurate translations.
    *   Quality Control: Fine-tune translations with options for base prompts, character introductions, background settings, and translation styles.  Includes features for one-click AI refinement, formatting, and terminology extraction.

### Additional Notes

</summary>

*   `Multiple Key Rotation`
    >   To use multiple keys for faster translation and share the consumption, use keys of the same type and separate them with commas (without line breaks).  For example: key1,key2,key3

*   `Batch File Translation`
    >   Place all files to be translated in the input folder; multi-folder structures are also supported.

*   `Configuration Migration`
    >   Configuration information is stored in resource/config.json. Copy this file to the new version's resource folder when updating.

## Contribution Guide

</summary>

*   **`Developing Enhanced Plugins`**: Follow the [plugin writing guide](https://github.com/NEKOparapa/AiNiee/blob/main/PluginScripts/README.md) to create powerful new plugins.

*   **`Improving or Adding File Support`**: Requires coding skills. Modify the source code by pulling the source code. File reading/writing code is located in the ModuleFolders\FileReader and FileOutputer folders. [Writer System Guide](https://github.com/NEKOparapa/AiNiee/blob/main/ModuleFolders/FileAccessor/README.md).  UI support in UserInterface\Setting's ProjectSettingsPage.

*   **`Improving Regular Expression Library`**: Enhancements to the regex library will greatly benefit in-game embedding and future translation work. The regex library is located in the [Resource\Regex](https://github.com/NEKOparapa/AiNiee/blob/main/Resource/Regex/regex.json) folder.

*   **`Improving Interface Translation`**: If the UI text translation is inaccurate, submit your suggestions or modify it directly. Localized text is in the [Resource\Localization](https://github.com/NEKOparapa/AiNiee/tree/main/Resource/Localization) folder.
