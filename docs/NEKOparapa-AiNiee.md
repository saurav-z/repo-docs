## üöÄ Translate Anything with AiNiee: Your All-in-One AI Translation Tool

AiNiee is a powerful AI-powered translation tool designed to effortlessly translate a wide range of text formats, including games, books, subtitles, and documents. [Explore the AiNiee repository on GitHub](https://github.com/NEKOparapa/AiNiee) to get started!

## ‚ú® Key Features:

*   üéÆ **Game Translation Powerhouse:** Seamlessly supports popular game text extraction tools like Mtool, Translator++, and more.
*   üìö **Versatile Format Support:** Handles various formats including I18Next data, Epub/TXT ebooks, Srt/Vtt/Lrc subtitles, Word/PDF/MD documents, and more.
*   ‚è±Ô∏è **Fast & Efficient:** Translate complex texts in a fraction of the time, with one-click file recognition and language detection.
*   üéØ **Advanced Long-Text Optimization:** Utilizes advanced techniques such as lightweight translation formatting, thought chain translation, AI terminology, and context association for accuracy.

* `Multi-Key Rotation`
  >  To speed up translations with multiple API keys, use keys from the same account type, separated by commas (no line breaks), e.g., key1,key2,key3.

* `Batch File Translation`
  > Place all files you need translated in the input folder. Supports multiple folder structures.

* `Configuration Migration`
  > Configuration information is stored in `resource/config.json`.  Copy this file to the new version's `resource` folder when updating.

## ü§ù Contributing

<details>
<summary>
  
### Contribution Guidelines
</summary>

*   **`Develop Enhancement Plugins`**:  Follow the [plugin development guide](https://github.com/NEKOparapa/AiNiee/blob/main/PluginScripts/README.md) to create enhanced plugins.
*   **`Improve or Add File Support`**: Requires coding skills; modify the source code. The file reading code is in the ModuleFolders\FileReader and FileOutputer folders. See the [reader/writer system guide](https://github.com/NEKOparapa/AiNiee/blob/main/ModuleFolders/FileAccessor/README.md). UI support in UserInterface\Setting of the ProjectSettingsPage.
*   **`Refine the Regular Expression Library`**: Completing the regular expression library greatly assists game integration and benefits the translation work and other users. The regular expression library is in the [Resource\Regex](https://github.com/NEKOparapa/AiNiee/blob/main/Resource/Regex/regex.json) folder.
*   **`Improve UI Translation`**: UI text may need better translation. Submit suggestions or modify directly. Localized text in the [Resource\Localization](https://github.com/NEKOparapa/AiNiee/tree/main/Resource/Localization) folder.
