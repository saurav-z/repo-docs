## üîë Key Features

*   üéÆ **Game Translation Mastery**: Extensive support for popular game text extraction tools like Mtool, Translator++, ParaTranzr, and more, ensuring seamless game translation.
*   üìö **Versatile Format Support**: Handle a wide array of file formats including I18Next data, Epub/TXT ebooks, Srt/Vtt/Lrc subtitles, Word/PDF/MD documents, and more.
*   ‚è±Ô∏è **Intelligent & Efficient**: One-click operation automatically detects files and languages, offering incredibly fast translation speeds.
*   üéØ **Advanced Translation Quality**: Utilize techniques such as light translation formats, chain-of-thought translation, AI terminology tables, and context association for accurate and coherent results in long-form texts.
*   üíé **Customization for Excellence**: Fine-tune your translations with prompts for basic instructions, character introductions, background settings, and translation styles. Also includes AI polishing, AI layout, and terminology extraction features for superior quality.

### üí° Additional Information
</summary>

* `Multiple Key Rotation`
  > To use multiple keys to share the consumption load, use keys of the same type. Separate each key with an English comma without line breaks. For example: key1,key2,key3

* `Batch File Translation`
  > Simply place all files needing translation in the input folder, which also supports multi-folder structures.

* `Configuration Migration`
  > Configuration information is stored in resource/config.json. You can copy this file to the resource folder of a new version after downloading it.
