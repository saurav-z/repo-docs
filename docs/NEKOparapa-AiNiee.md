## üöÄ AiNiee: Your All-in-One AI Translation Solution for Games, Books, and More!

AiNiee is a powerful AI-powered translation tool designed to effortlessly translate complex text, including games, books, subtitles, and documents. This README provides a comprehensive overview of its features, usage, and contribution guidelines. Check out the original repo [here](https://github.com/NEKOparapa/AiNiee).

---

## üîë Key Features

*   **üåê Broad Format Support:**

    *   üéÆ **Game Translation:** Deeply supports game text extraction tools like Mtool, Renpy, Translator++, ParaTranzr, VNText, and SExtractor.
    *   üìö **Diverse Compatibility:** Seamlessly handles I18Next data, Epub/TXT e-books, Srt/Vtt/Lrc subtitles, Word/PDF/MD documents, and more.

*   **üí° Smart & Efficient:**

    *   üöÄ **One-Click Operation:** Simply drag and drop, and AiNiee automatically identifies files and languages, eliminating the need for manual settings.
    *   ‚è±Ô∏è **Lightning-Fast Translations:** Get your translated text in a matter of minutes.

*   **‚ú® Optimized for Quality:**

    *   üéØ **Overcomes Limitations:** Utilizes techniques like lightweight translation formats, thought chain translation, AI terminology tables, and contextual associations to ensure accurate and coherent translations of long texts.
    *   üíé **Quality Enhancement:** Supports prompt customization (e.g., basic prompts, character introductions, background settings, translation styles) and features like one-click AI refinement, one-click AI formatting, and one-click terminology extraction for users who demand higher translation quality.

## ü§ù Contribution Guide

</summary>

*   **`Develop Enhanced Plugins`**: Please follow the [Plugin Writing Guide](https://github.com/NEKOparapa/AiNiee/blob/main/PluginScripts/README.md) to develop more powerful function plugins.

*   **`Improve or Add Support for Files`**: Requires some code programming skills; pull the source code for improvement. The file reading code is located in the ModuleFolders\FileReader and FileOutputer folders. [Writing Guide for Readers and Writers](https://github.com/NEKOparapa/AiNiee/blob/main/ModuleFolders/FileAccessor/README.md). UI support is located in the UserInterface\Setting of the ProjectSettingsPage.

*   **`Improve Regex Library`**: The completeness of the Regex library will greatly assist in the embedded work in the game, benefit the next game translation work and benefit other translation users; the Regex library is in the [Resource\Regex](https://github.com/NEKOparapa/AiNiee/blob/main/Resource/Regex/regex.json) folder.

*   **`Improve Interface Translation`**: UI text in multiple languages may not be accurately or appropriately translated. You can submit your modification opinions or modify them directly. Localized text is located in the [Resource\Localization](https://github.com/NEKOparapa/AiNiee/tree/main/Resource/Localization) folder.
