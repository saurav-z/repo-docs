    *   üéÆ **Game Translation:** Supports Mtool, Renpy, Translator++, ParaTranzr, VNText, SExtractor, and more.

*   ‚è±Ô∏è **Intelligent & Efficient:**

    >  To distribute the processing load using multiple keys (use keys from the same account type), enter them separated by commas without line breaks. Example: key1,key2,key3

    > Place all files to be translated in the input folder. Supports multi-folder structures.

*   `Configuration Migration`
    > Configuration information is stored in `resource/config.json`. Copy this file to the `resource` folder of the new version.

## ü§ù Contributing to AiNiee

</summary>

*   **`Enhance with Plugins`**:  Develop powerful plugins following the [Plugin Writing Guide](https://github.com/NEKOparapa/AiNiee/blob/main/PluginScripts/README.md).
*   **`Improve File Support`**:  Contribute by modifying the code. The file reading code is located in the `ModuleFolders\FileReader` and `FileOutputer` folders. Refer to the [Reader/Writer System Guide](https://github.com/NEKOparapa/AiNiee/blob/main/ModuleFolders/FileAccessor/README.md).  UI support is in `UserInterface\Setting` of the `ProjectSettingsPage`.
*   **`Enhance Regular Expression Library`**: Improve the regex library (located in `Resource\Regex\regex.json`). This will greatly help with in-game embedding work and benefit other translation users.
*   **`Improve UI Translation`**: If the UI text isn't accurate or appropriate, submit your suggestions or make edits directly. Localized text is in the `Resource\Localization` folder.
