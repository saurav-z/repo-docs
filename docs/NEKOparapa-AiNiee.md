## AiNiee: The Ultimate AI Translation Tool for Games, Books, and More

**Tired of struggling with clunky translations?** AiNiee provides seamless, AI-powered translation for games, books, subtitles, and documents.

### Key Features:

*   üéÆ **Game Translation Mastery:** Deep support for popular game text extraction tools like Mtool, Translator++, and more, ensuring effortless game localization.
*   üìö **Broad Format Support:**  Handles a wide array of formats, including I18Next data, Epub/TXT ebooks, SRT/VTT/LRC subtitles, and Word/PDF/MD documents.
*   üöÄ **Effortless & Speedy:** One-click operation automatically detects files and languages, delivering translations in minutes.
*   üéØ **Enhanced Quality for Long Texts:** Leverages cutting-edge techniques like lightweight translation formats, chain-of-thought translation, AI terminology tables, and contextual association to ensure accuracy and coherence.
*   üíé **Fine-Tuning for Superior Results:** Offers options to customize translations with prompts for basic text, character introductions, background settings, and translation styles. Includes AI polishing, formatting, and terminology extraction features.

### Other Information

</summary>

*   `Multiple Key Rotation`
    >  To use multiple keys to reduce consumption pressure and speed up translation, use keys of the same type, separating them with commas without line breaks. Example: key1,key2,key3

*   `Batch File Translation`
    > Place all files for translation in the input folder, supporting multi-folder structures.

*   `Configuration Migration`
    > Configuration information is stored in resource/config.json. Copy it to the new version's resource folder when updating.

## Contributing

<details>
<summary>
  
### Contribution Guidelines

</summary>

*   **`Develop Enhanced Plugins`**: Follow the [plugin development guide](https://github.com/NEKOparapa/AiNiee/blob/main/PluginScripts/README.md) to create powerful new plugins.

*   **`Improve or Add Support for Files`**: Requires coding knowledge. Pull the source code and make improvements. The file reading code is in the ModuleFolders\FileReader and FileOutputer folders.  The UI support is in UserInterface\Setting\ProjectSettingsPage.  See the [Reader/Writer System Guide](https://github.com/NEKOparapa/AiNiee/blob/main/ModuleFolders/FileAccessor/README.md).

*   **`Enhance the Regular Expression Library`**:  A complete regex library will greatly assist in game integration.  The regex library is in the [Resource\Regex](https://github.com/NEKOparapa/AiNiee/blob/main/Resource/Regex/regex.json) folder.

*   **`Improve UI Translations`**:  If the multi-language UI text is not accurate or appropriate, submit suggestions or modify directly. The localization text is in the [Resource\Localization](https://github.com/NEKOparapa/AiNiee/tree/main/Resource/Localization) folder.

</details>
