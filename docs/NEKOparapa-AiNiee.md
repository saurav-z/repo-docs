## Unleash the Power of AI Translation with AiNiee!

AiNiee is a powerful, AI-driven translation tool designed to effortlessly translate complex text content like games, books, subtitles, and documents. [**Check out the original repo here!**](https://github.com/NEKOparapa/AiNiee)

---

## Key Features

*   **Versatile Format Support**:
    *   üéÆ **Game Translation**: Deeply supports game text extraction tools like Mtool, Renpy, Translator++, ParaTranzr, VNText, and SExtractor.
    *   üìö **Broad Compatibility**: Handles I18Next data, Epub/TXT ebooks, Srt/Vtt/Lrc subtitles, Word/PDF/MD documents, and more.

*   **Intelligent and Efficient**:
    *   üöÄ **One-Click Operation**: Simply drag and drop; automatic file and language detection eliminates the need for manual configuration.
    *   ‚è±Ô∏è **Lightning-Fast Translation**: Get your translations in minutes, saving you valuable time.

*   **Optimized for Long Texts and High Quality**:
    *   üéØ **Overcoming Limitations**: Uses techniques like light translation formats, chain-of-thought translation, AI terminology tables, and context association to ensure coherence and accuracy for long texts.
    *   üíé **Quality Enhancement**: Supports prompt adjustments for basic prompts, character introductions, background settings, and translation styles. Includes one-click AI refinement, formatting, and terminology extraction for superior quality.

*   `Multi-Key Rotation`
    >   To accelerate translation using multiple keys, use keys from the same account type and separate them with commas (no line breaks). For example: key1,key2,key3
*   `Batch File Translation`
    >   Place all files needing translation into the input folder; multiple folder structures are also supported.
*   `Configuration Migration`
    >   Configuration information is stored in `resource/config.json`. Copy this file to the new version's resource folder when updating.

## Contribution Guide
</summary>

*   **`Develop Enhancement Plugins`**: Develop powerful feature plugins based on the [Plugin Writing Guide](https://github.com/NEKOparapa/AiNiee/blob/main/PluginScripts/README.md).

*   **`Improve or Add Support for Files`**: Requires some coding skills; pull the source code for improvements. The file reading code is in the `ModuleFolders\FileReader` and `FileOutputer` folders. See the [Reader/Writer System Writing Guide](https://github.com/NEKOparapa/AiNiee/blob/main/ModuleFolders/FileAccessor/README.md). UI support is in `UserInterface\Setting\ProjectSettingsPage`.

*   **`Improve Regular Expression Library`**: Improving the regex library will greatly aid in-game embedded work, benefiting future game translation and other translation users. The regex library is in the `Resource\Regex` folder.

*   **`Improve UI Translation`**: UI text in multiple languages might need more accurate translation; submit your suggestions or modify directly. Localized text is in the `Resource\Localization` folder.
