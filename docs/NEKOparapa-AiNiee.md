# AiNiee: The Ultimate AI Translation Tool for Games, Books, and More!

Effortlessly translate complex text formats like games, books, and subtitles with AiNiee.  Dive into the original project on [GitHub](https://github.com/NEKOparapa/AiNiee).

## Key Features

*   **Versatile Format Support:**
    *   üéÆ **Game Translation:** Deep support for Mtool, Renpy, Translator++, ParaTranzr, VNText, SExtractor, and more.
    *   üìö **Broad Compatibility:** Handles I18Next data, Epub/TXT ebooks, Srt/Vtt/Lrc subtitles, Word/PDF/MD documents, and more.

*   **Smart & Efficient:**
    *   üöÄ **One-Click Translation:**  Drag, drop, and AiNiee automatically detects files and languages, simplifying the process.
    *   ‚è±Ô∏è **Lightning-Fast:** Get your translations done in the time it takes to enjoy a soda.

*   **High-Quality, Long-Text Optimized:**
    *   üéØ **Overcomes Limitations:** Leverages technologies like light translation formats, chain-of-thought translation, AI terminology tables, and context awareness to ensure fluency and accuracy.
    *   üíé **Quality Control:** Includes features like AI polishing, AI typesetting, and terminology extraction, supporting advanced users seeking superior translation quality with prompt adjustment (Basic prompts, Character introductions, Background settings, Translation styles).

### Additional Notes
</summary>

*   `Multi-Key Polling`
    > If you want to use multiple keys to share the consumption pressure, please use keys of the same type of account, and add an English comma between each key, without line breaks. For example: key1,key2,key3

*   `Batch File Translation`
    > Just put all the files that need to be translated in the input folder, and it also supports the multi-folder structure

*   `Configuration Migration`
    > Configuration information is stored in resource's config.json, you can copy it to the resource of the new version when downloading a new version.

## Contributing

<details>
<summary>
  
### How to Contribute
</summary>

*   **`Develop Enhanced Plugins`**: Develop more powerful feature plugins based on the [Plugin Writing Guide](https://github.com/NEKOparapa/AiNiee/blob/main/PluginScripts/README.md)
*   **`Improve or Add Supported Files`**: Requires some code programming ability, pull the source code to improve. The specific code for reading files is in the ModuleFolders\\FileReader and FileOutputer folders. [Reader/Writer System Writing Guide](https://github.com/NEKOparapa/AiNiee/blob/main/ModuleFolders/FileAccessor/README.md). UI support is in the UserInterface\Setting's ProjectSettingsPage.
*   **`Improve Regular Expression Library`**: The completeness of the regular expression library will greatly help the embedded work in the game, and benefit the next game translation work and benefit other translation users. The regular expression library is in the [Resource\\Regex](https://github.com/NEKOparapa/AiNiee/blob/main/Resource/Regex/regex.json) folder
*   **`Improve UI Translation`**: The UI text of multilingual interfaces may not be translated accurately or appropriately, you can submit your modification opinions, or modify it directly. Localized text is in the [Resource\\Localization](https://github.com/NEKOparapa/AiNiee/tree/main/Resource/Localization) folder

</details>
