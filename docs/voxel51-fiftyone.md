# FiftyOne: The Open-Source Toolkit for Computer Vision ðŸš€

**FiftyOne empowers you to build superior computer vision models by providing tools for dataset creation, model analysis, and data quality improvement.**

[Visit the original repo on GitHub](https://github.com/voxel51/fiftyone) for more information and to get started.

## Key Features

*   **Visualize and Explore Datasets:** Easily inspect and interact with your image, video, and label data.

*   **Advanced Model Analysis:** Evaluate model performance, pinpoint failure points, and refine your models.

*   **Interactive Embedding Exploration:** Select points of interest and view the corresponding samples and labels.

*   **Streamlined Data Curation:** Quickly identify and correct data inconsistencies, annotation errors, and edge cases.

*   **Seamless Integrations:** Compatible with popular deep learning frameworks, including PyTorch, Hugging Face, and Ultralytics.

*   **Extensible and Customizable:** Adapt FiftyOne to suit your specific computer vision project needs with plugins and customization options.

## Installation

```bash
pip install fiftyone
```
