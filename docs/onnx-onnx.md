# ONNX: Open Neural Network Exchange

**ONNX empowers AI developers by providing an open standard for AI models, enabling seamless interoperability and accelerating innovation.** Find the original repo [here](https://github.com/onnx/onnx).

## Key Features

*   **Open Format:** Defines a standard format for AI models, supporting both deep learning and traditional ML.
*   **Extensible Computation Graph:** Provides a flexible and extensible graph model for representing AI computations.
*   **Built-in Operators:** Includes a comprehensive set of pre-defined operators for common AI tasks.
*   **Framework Interoperability:** Facilitates the exchange of models between different AI frameworks, tools, and hardware platforms.
*   **Focus on Inferencing:** Primarily designed for optimizing and deploying models for inference (scoring) in production environments.
