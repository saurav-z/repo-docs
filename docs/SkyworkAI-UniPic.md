## Key Features of Skywork UniPic

Skywork UniPic empowers users with a suite of innovative features:

*   **High-Fidelity Text-to-Image Generation:** Generate stunning images directly from natural language descriptions.
*   **Advanced Image Editing Capabilities:** Perform seamless inpainting, outpainting, and object manipulation for creative control.
*   **Robust Image Understanding:** Leverage powerful perception capabilities for a variety of visual tasks.
*   **Efficient Architecture:** Benefit from models optimized for both high accuracy and efficient deployment.

---

## UniPic Model Series:

This repository features two distinct modeling paradigms:

*   **UniPic-1.0:** Unified Autoregressive Modeling (1.5B parameters) for joint visual understanding and generation, enabling a single transformer to handle both perception and synthesis tasks.
*   **UniPic-2.0 Series:** Featuring SD3.5M-Kontext and MetaQuery variants, built on efficient architectures with diffusion post-training, delivering state-of-the-art results.
