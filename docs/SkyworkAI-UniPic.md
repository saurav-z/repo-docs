## What is Skywork UniPic?

Skywork UniPic is a collection of unified multimodal models designed to excel in various image-related tasks. This repository provides the model weights and official implementations for both UniPic-1.0 and the UniPic-2.0 series.  Visit the [original repository](https://github.com/SkyworkAI/UniPic) for complete details.

## Key Features

*   **Text-to-Image Generation:** Generate stunning, high-fidelity images from textual descriptions.
*   **Image Editing:** Perform seamless inpainting, outpainting, and object manipulation with ease.
*   **Image Understanding:**  Gain robust perception capabilities for a wide range of visual tasks.
*   **Efficient Architecture:** Benefit from models optimized for both accuracy and efficient deployment.
*   **Unified Modeling Paradigms:** Explore innovative approaches, including unified autoregressive modeling (UniPic-1.0) and efficient architectures with diffusion post-training (UniPic-2.0).

## Model Variants

*   **UniPic-1.0:**  A 1.5B parameter model employing **Unified Autoregressive Modeling** for joint visual understanding and generation.
*   **UniPic-2.0 Series:** Features *SD3.5M-Kontext* and *MetaQuery* variants, built on **Efficient Architectures** with **Diffusion Post-Training**, offering state-of-the-art performance.
