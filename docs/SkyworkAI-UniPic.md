## Key Features of Skywork UniPic

*   **Text-to-Image Generation:** Transform natural language prompts into stunning, high-fidelity images.
*   **Image Editing:** Enjoy seamless inpainting, outpainting, and object manipulation with ease.
*   **Image Understanding:** Leverage robust perception capabilities for a wide range of visual tasks.
*   **Efficient Architecture:** Benefit from optimized architectures designed for both exceptional accuracy and efficient deployment.

## UniPic Models Overview

Skywork UniPic offers two powerful modeling paradigms:

*   **UniPic-1.0:** This model features a **Unified Autoregressive Modeling** approach with 1.5 billion parameters, enabling joint visual understanding and generation within a single transformer.
*   **UniPic-2.0 Series:** These models utilize **Efficient Architectures** enhanced with **Diffusion Post-Training**. Variants include *SD3.5M-Kontext* and *MetaQuery*, delivering state-of-the-art performance in text-to-image generation, fine-grained image editing, and multimodal reasoning.
