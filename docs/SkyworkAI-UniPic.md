## About UniPic

This repository provides access to the **Skywork-UniPic** suite of unified multimodal models. UniPic offers cutting-edge solutions for image understanding, generation, and editing, leveraging two distinct modeling paradigms.  Explore the potential of AI-powered image manipulation and understanding with our state-of-the-art models.

**[Explore the original UniPic repository on GitHub](https://github.com/SkyworkAI/UniPic)**

## Key Features

*   **Text-to-Image Generation:** Create stunning images from textual descriptions, bringing your ideas to life.
*   **Advanced Image Editing:** Perform seamless inpainting, outpainting, and object manipulation for refined visual content.
*   **Robust Image Understanding:** Enable powerful perception capabilities, unlocking new possibilities for visual tasks.
*   **Efficient Architecture:** Experience optimized performance and deployability, ensuring accessibility and speed.
*   **Unified Modeling:** Benefit from models that excel at both perception and synthesis tasks.

## Model Versions

UniPic offers two primary model families, each with unique strengths:

*   **UniPic-1.0:** A 1.5B parameter model based on **Unified Autoregressive Modeling**, excelling at joint visual understanding and generation.
*   **UniPic-2.0 Series:** Featuring *SD3.5M-Kontext* and *MetaQuery* variants, built on **Efficient Architectures** with **Diffusion Post-Training**, delivering leading performance in text-to-image generation, fine-grained image editing, and multimodal reasoning.
