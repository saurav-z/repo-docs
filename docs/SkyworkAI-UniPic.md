Create, understand, and edit images with unprecedented ease using **Skywork UniPic**, a revolutionary suite of unified multimodal models. ([Original Repository](https://github.com/SkyworkAI/UniPic))

## Key Features

*   **Advanced Image Generation:** Generate stunning, high-fidelity images from text descriptions.
*   **Intuitive Image Editing:** Effortlessly perform inpainting, outpainting, and object manipulation for creative control.
*   **Robust Image Understanding:** Gain deep insights into visual content with powerful perception capabilities.
*   **Efficient Architecture:** Optimized for both cutting-edge performance and practical deployment across diverse platforms.

## Overview of UniPic Models

This repository provides model weights and implementations for the following:

*   **UniPic-1.0:** A 1.5B parameter model utilizing **Unified Autoregressive Modeling** for both image understanding and generation. It employs a single transformer architecture for perception and synthesis.
*   **UniPic-2.0 Series:** Featuring *SD3.5M-Kontext* and *MetaQuery* variants, these models leverage **Efficient Architectures** with **Diffusion Post-Training** for state-of-the-art results in text-to-image generation, fine-grained image editing, and multimodal reasoning.
