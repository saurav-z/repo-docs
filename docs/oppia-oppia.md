# Oppia: Interactive Learning for Everyone

**Oppia is an open-source online learning platform designed to create and share engaging, interactive educational activities.** ([View the original repository](https://github.com/oppia/oppia))

[![Full-stack tests](https://github.com/oppia/oppia/actions/workflows/full_stack_tests.yml/badge.svg)](https://github.com/oppia/oppia/actions/workflows/full_stack_tests.yml)

Oppia transforms the way people learn with interactive explorations that simulate one-on-one tutoring, fostering active learning and providing immediate feedback. The platform empowers educators and learners to create and share educational content.

**Key Features:**

*   **Interactive Explorations:** Build engaging lessons that feel like a conversation with a tutor.
*   **Community-Driven:** Benefit from a platform built by and for the community, welcoming contributors of all backgrounds.
*   **Free Educational Resources:** Access free lessons on mathematics, and find more educational offerings planned.
*   **Open Source:** Contribute to and customize the platform built with Python, Angular, and Google App Engine.
