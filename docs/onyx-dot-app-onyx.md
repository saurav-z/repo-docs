**[Onyx](https://www.onyx.app/)** (formerly Danswer) empowers your team with AI-powered search and knowledge management.  Integrate Onyx with your existing tools for a seamless experience.

## Key Features

*   **AI-Powered Chat:** Engage with any LLM of your choice through a feature-rich chat interface.
*   **Extensive Connectors:** Sync knowledge and access controls across 40+ connectors, including Google Drive, Slack, Confluence, and Salesforce.
*   **Custom AI Agents:** Create AI agents with unique prompts, knowledge, and actions to automate tasks.
*   **Flexible Deployment:** Deploy securely on your laptop, on-premise, or in the cloud for any scale.
*   **Deep Research:** Utilize custom deep learning models for advanced indexing and inference, improving search accuracy.
*   **Knowledge Curation:** Leverage document sets, query history, and usage analytics to optimize your knowledge base.
*   **Secure & Scalable:** Benefit from flexible security features like SSO (OIDC/SAML/OAuth2), RBAC, and scalable deployment options.

## Deployment

Get started in seconds with **[Onyx Cloud](https://cloud.onyx.app/signup)** for a free trial.

Onyx can also be run locally or deployed on a virtual machine using a single `docker compose` command.  See the [documentation](https://docs.onyx.app/quickstart) for details.  Onyx also supports high-availability and scalable deployments on Kubernetes. Reference [here](https://github.com/onyx-dot-app/onyx/tree/main/deployment).
