## Deployment

Get started with Onyx in seconds! Try **[Onyx Cloud](https://cloud.onyx.app/signup)** for a free, easy-to-use experience.

For self-hosting, deploy Onyx locally or on a virtual machine using a simple `docker compose` command. See our [documentation](https://docs.onyx.app/quickstart) for detailed instructions. We also support high-availability and scalable deployments on Kubernetes.  See our [deployment documentation](https://github.com/onyx-dot-app/onyx/tree/main/deployment) for more information.

## Additional Benefits

*   **Custom Models:**  Leverage Onyx's custom deep learning models for superior indexing and inference, constantly improving based on user feedback.
*   **Robust Security:** Benefit from flexible security features, including SSO (OIDC/SAML/OAuth2), RBAC, and credential encryption.
*   **Knowledge Curation:** Utilize features like document sets, query history, and usage analytics for enhanced knowledge management.
*   **Scalability:** Proven to scale for tens of thousands of users and millions of documents.
