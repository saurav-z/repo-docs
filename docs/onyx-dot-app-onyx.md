Onyx (formerly Danswer) is an open-source platform that connects to your company's data, apps, and people, providing a powerful AI-driven search and knowledge management solution.  **[Explore the Onyx GitHub Repository](https://github.com/onyx-dot-app/onyx)**.

## Key Features

*   **AI-Powered Chat Interface:** Interact with your data using a user-friendly chat interface, leveraging the power of Gen-AI models.
*   **Versatile LLM Integration:**  Connect Onyx with your preferred Large Language Models (LLMs) to optimize search results and answer generation.
*   **Seamless Data Integration:** Connect and synchronize knowledge and access controls across 40+ connectors, including:
    *   Google Drive
    *   Confluence
    *   Slack
    *   Salesforce
    *   And many more (see full list below).
*   **Custom AI Agents:**  Create AI agents with custom prompts, knowledge, and actions to automate tasks and streamline workflows.
*   **Flexible Deployment:** Deploy Onyx securely on your laptop, on-premise servers, or in the cloud.
*   **Deep Learning Models:** Utilizing custom deep learning models for indexing and inference.
*   **Security Features:** Flexible security features like SSO (OIDC/SAML/OAuth2), RBAC, and encryption of credentials.
*   **Knowledge Curation:** Features like document-sets, query history, usage analytics, and more.
*   **Scalable Deployment:** Tested and proven deployment options for scalability, including Kubernetes.

## Deployment

Get started in seconds with **[Onyx Cloud](https://cloud.onyx.app/signup)** for a free trial.

Onyx can also be deployed locally using a single `docker compose` command or on a virtual machine.  Refer to the [Onyx documentation](https://docs.onyx.app/quickstart) for detailed instructions. High-availability and scalable deployments on Kubernetes are also supported; see the [deployment documentation](https://github.com/onyx-dot-app/onyx/tree/main/deployment).

## Other Notable Benefits of Onyx

*   Custom deep learning models for indexing and inference time, only through Onyx + learning from user feedback.
*   Flexible security features like SSO (OIDC/SAML/OAuth2), RBAC, encryption of credentials, etc.
*   Knowledge curation features like document-sets, query history, usage analytics, etc.
*   Scalable deployment options tested up to many tens of thousands users and hundreds of millions of documents.
