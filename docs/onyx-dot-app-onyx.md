## Deployment

**Get started in seconds!** Try it for free with [Onyx Cloud](https://cloud.onyx.app/signup).

Onyx can be run locally (even on a laptop) or deployed on a virtual machine with a single `docker compose` command. Check out our [docs](https://docs.onyx.app/quickstart) to learn more.

We offer built-in support for high-availability/scalable deployment on Kubernetes.  See [here](https://github.com/onyx-dot-app/onyx/tree/main/deployment).

## Additional Benefits

*   Custom deep learning models for indexing and inference, enhanced through user feedback.
*   Flexible security features including SSO (OIDC/SAML/OAuth2), RBAC, and credential encryption.
*   Knowledge curation features like document sets, query history, and usage analytics.
*   Scalable deployment options tested for tens of thousands of users and millions of documents.
