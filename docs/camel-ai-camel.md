## Key Features of CAMEL

*   ‚úÖ **Large-Scale Agent Systems:** Simulate up to 1 million agents to study emergent behaviors and scaling laws.
*   ‚úÖ **Dynamic Communication:** Enable real-time interactions for complex task collaboration.
*   ‚úÖ **Stateful Memory:** Equip agents with memory for improved decision-making over extended interactions.
*   ‚úÖ **Multiple Benchmarks:** Evaluate agent performance with standardized benchmarks for reproducible results.
*   ‚úÖ **Diverse Agent Types:** Support a variety of roles, tasks, models, and environments for interdisciplinary research.
*   ‚úÖ **Data Generation and Tool Integration:** Automate dataset creation and integrate seamlessly with various tools.

## CAMEL Framework Design Principles

*   **üß¨ Evolvability:**  The framework supports continuous evolution through data generation and environment interaction, driven by reinforcement or supervised learning.
*   **üìà Scalability:** Designed to handle millions of agents, ensuring efficient coordination, communication, and resource management.
*   **üíæ Statefulness:** Agents maintain stateful memory, allowing for multi-step interactions and complex task handling.
*   **üìñ Code-as-Prompt:**  Code and comments serve as prompts, promoting clarity and interpretability for both humans and agents.

CAMEL empowers researchers with a comprehensive platform for multi-agent systems research.  Join our vibrant community of 100+ researchers and leverage CAMEL's features to:

## Quick Start - Get Started with CAMEL!

CAMEL is easily installed via PyPI:

```bash
pip install camel-ai
```

### Example:  Creating a ChatAgent

1.  **Install web tools (if needed):**
    ```bash
    pip install 'camel-ai[web_tools]'
    ```
2.  **Set your OpenAI API key:**
    ```bash
    export OPENAI_API_KEY='your_openai_api_key'
    ```
3.  **Run this Python code:**
    ```python
    from camel.models import ModelFactory
    from camel.types import ModelPlatformType, ModelType
    from camel.agents import ChatAgent
    from camel.toolkits import SearchToolkit

    model = ModelFactory.create(
      model_platform=ModelPlatformType.OPENAI,
      model_type=ModelType.GPT_4O,
      model_config_dict={"temperature": 0.0},
    )

    search_tool = SearchToolkit().search_duckduckgo

    agent = ChatAgent(model=model, tools=[search_tool])

    response_1 = agent.step("What is CAMEL-AI?")
    print(response_1.msgs[0].content)
    # CAMEL-AI is the first LLM (Large Language Model) multi-agent framework
    # and an open-source community focused on finding the scaling laws of agents.
    # ...

    response_2 = agent.step("What is the Github link to CAMEL framework?")
    print(response_2.msgs[0].content)
    # The GitHub link to the CAMEL framework is
    # [https://github.com/camel-ai/camel](https://github.com/camel-ai/camel).
    ```

Explore the  [installation section](https://github.com/camel-ai/camel/blob/master/docs/get_started/installation.md) for detailed instructions. Dive into our [Cookbooks](https://docs.camel-ai.org) to build powerful multi-agent systems.
