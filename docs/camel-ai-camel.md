## Key Features of the CAMEL Framework

*   ‚úÖ **Evolvability:** Enable continuous agent evolution through data generation and environment interaction.
*   ‚úÖ **Scalability:** Support massive multi-agent systems, efficiently managing coordination, communication, and resources.
*   ‚úÖ **Statefulness:** Equip agents with memory to perform multi-step interactions and tackle complex tasks.
*   ‚úÖ **Code-as-Prompt:** Utilize clear, readable code to guide agent behavior and facilitate understanding.

<br>

## Why Choose CAMEL for Your Research?

CAMEL is a community-driven research platform for multi-agent systems, trusted by researchers worldwide.  Here's why:

*   ‚úÖ **Large-Scale Agent Systems:** Simulate up to 1 million agents to study emergent behaviors and scaling laws.
*   ‚úÖ **Dynamic Communication:** Facilitate real-time interaction and collaboration among agents.
*   ‚úÖ **Stateful Memory:** Empower agents with historical context for improved decision-making.
*   ‚úÖ **Comprehensive Benchmarks:** Evaluate agent performance with standardized benchmarks.
*   ‚úÖ **Diverse Agent Types:** Experiment with various roles, tasks, models, and environments.
*   ‚úÖ **Data Generation & Tool Integration:** Automate dataset creation and streamline research workflows.

## Getting Started: Quick Installation and Example

Install CAMEL quickly using pip:

```bash
pip install camel-ai
```

### Example: ChatAgent with DuckDuckGo Search

1.  **Install web tools:**

    ```bash
    pip install 'camel-ai[web_tools]'
    ```

2.  **Set your OpenAI API key:**

    ```bash
    export OPENAI_API_KEY='your_openai_api_key'
    ```

3.  **Run the Python code:**

    ```python
    from camel.models import ModelFactory
    from camel.types import ModelPlatformType, ModelType
    from camel.agents import ChatAgent
    from camel.toolkits import SearchToolkit

    model = ModelFactory.create(
      model_platform=ModelPlatformType.OPENAI,
      model_type=ModelType.GPT_4O,
      model_config_dict={"temperature": 0.0},
    )

    search_tool = SearchToolkit().search_duckduckgo

    agent = ChatAgent(model=model, tools=[search_tool])

    response_1 = agent.step("What is CAMEL-AI?")
    print(response_1.msgs[0].content)

    response_2 = agent.step("What is the Github link to CAMEL framework?")
    print(response_2.msgs[0].content)
    ```

For detailed installation and configuration, see the [installation section](https://github.com/camel-ai/camel/blob/master/docs/get_started/installation.md).
