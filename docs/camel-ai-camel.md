## CAMEL: Explore the Scaling Laws of AI Agents

**CAMEL is an open-source framework for building and studying multi-agent systems, driving innovation in AI agent capabilities.**  [Explore the original repository](https://github.com/camel-ai/camel).

<p style="line-height: 1.5; text-align: center;"> üê´ CAMEL (Communicative Agents for Mind Exploration of Large Language Model Society) is an open-source community dedicated to finding the scaling laws of agents. We believe that studying these agents on a large scale offers valuable insights into their behaviors, capabilities, and potential risks. To facilitate research in this field, we implement and support various types of agents, tasks, prompts, models, and simulated environments.</p>

## Key Features

*   ‚úÖ  **Large-Scale Agent Systems:** Simulate up to 1 million agents for emergent behavior analysis and scaling law research.
*   ‚úÖ  **Dynamic Communication:** Enable real-time agent interactions for seamless task collaboration.
*   ‚úÖ  **Stateful Memory:** Equip agents with historical context for improved decision-making.
*   ‚úÖ  **Code-as-Prompt:**  Utilize clear, readable code as prompts for effective agent interpretation and operation.
*   ‚úÖ  **Extensible Design:**  Evolve your multi-agent systems continuously by generating data and interacting with the environment.
*   ‚úÖ  **Scalable Architecture:**  Support systems with millions of agents, with efficient coordination, communication, and resource management.
*   ‚úÖ  **Benchmarking Support:**  Evaluate agent performance using standard benchmarks for reproducibility and comparison.
*   ‚úÖ  **Versatile Agent Support:** Experiment with diverse roles, tasks, models, and environments for interdisciplinary research.
*   ‚úÖ  **Data Generation & Tool Integration:** Streamline research workflows with automated dataset creation and tool integration.

## Quick Start: Get Started with ChatAgent

Install CAMEL and its web tools dependencies:

```bash
pip install camel-ai[web_tools]
```

Set your OpenAI API key:

```bash
export OPENAI_API_KEY='your_openai_api_key'
```

Run the following Python code:

```python
from camel.models import ModelFactory
from camel.types import ModelPlatformType, ModelType
from camel.agents import ChatAgent
from camel.toolkits import SearchToolkit

model = ModelFactory.create(
    model_platform=ModelPlatformType.OPENAI,
    model_type=ModelType.GPT_4O,
    model_config_dict={"temperature": 0.0},
)

search_tool = SearchToolkit().search_duckduckgo

agent = ChatAgent(model=model, tools=[search_tool])

response_1 = agent.step("What is CAMEL-AI?")
print(response_1.msgs[0].content)

response_2 = agent.step("What is the Github link to CAMEL framework?")
print(response_2.msgs[0].content)
```

Explore our documentation at [docs.camel-ai.org](https://docs.camel-ai.org) to build powerful multi-agent systems.
For more detailed instructions, explore the [installation section](https://github.com/camel-ai/camel/blob/master/docs/get_started/installation.md).
