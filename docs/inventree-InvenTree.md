## InvenTree: Your Open-Source Inventory Management Solution

InvenTree is a powerful, open-source inventory management system designed to track and manage your parts and stock efficiently. Visit the [InvenTree repository](https://github.com/inventree/InvenTree) for more details.

**Key Features:**

*   **Comprehensive Inventory Tracking:** Manage parts, stock, and locations with ease.
*   **Web-Based Admin Interface:** Access and manage your inventory from anywhere.
*   **REST API:** Integrate with other applications using a robust REST API.
*   **Plugin System:** Extend functionality with custom plugins.
*   **Mobile App Support:** Access your inventory on the go with a companion mobile app.
*   **Extensive Integrations:** Integration with existing tools via API, Python module and plugins.
*   **Open Source & Community Driven:** Benefit from a collaborative and transparent development process.

## :star2: About the Project

InvenTree is a robust inventory management system offering powerful stock control and part tracking, with a Python/Django backend providing a web-based admin interface and a REST API. Explore its features and capabilities on [our website](https://inventree.org).

### :compass: Roadmap

Stay updated on our progress! Check out the [roadmap tag](https://github.com/inventree/InvenTree/issues?q=is%3Aopen+is%3Aissue+label%3Aroadmap) and [horizon milestone](https://github.com/inventree/InvenTree/milestone/42).

    <a href="https://docs.inventree.org/en/latest/start/docker/">Docker</a>
    <span> · </span>
    <a href="https://inventree.org/digitalocean"><img src="https://www.deploytodo.com/do-btn-blue-ghost.svg" alt="Deploy to DO" width="auto" height="40" /></a>
    <span> · </span>
    <a href="https://docs.inventree.org/en/latest/start/install/">Bare Metal</a>
</h4></div>

Install with a single line (see [the docs](https://docs.inventree.org/en/latest/start/installer/) for details):

```bash
wget -qO install.sh https://get.inventree.org && bash install.sh
```

Find comprehensive installation and setup instructions in the [getting started guide](https://docs.inventree.org/en/latest/start/install/).
