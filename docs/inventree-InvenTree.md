<!-- Introduction -->
## **InvenTree: The Open-Source Inventory Management System**

InvenTree is a powerful, open-source inventory management system designed to streamline stock control and part tracking for businesses and hobbyists alike.  ([View the original repository](https://github.com/inventree/InvenTree))

<!-- Key Features -->
## Key Features of InvenTree:

*   **Comprehensive Inventory Tracking:** Manage parts, stock levels, locations, and associated data with ease.
*   **Web-Based Interface:** Access your inventory from anywhere with a web browser.
*   **REST API:** Integrate InvenTree with other applications and systems via a robust REST API.
*   **Plugin System:** Extend functionality with custom plugins and integrations.
*   **Mobile App:**  Track and manage your inventory on the go with our companion mobile app.
*   **User-Friendly Interface:** An intuitive interface makes managing your inventory simple and efficient.
*   **Flexible Deployment:** Deploy InvenTree using Docker, or bare metal installation.
*   **Multi-Platform Support:** PostgreSQL, MySQL, and SQLite database support.

<!-- About the Project -->
## About InvenTree

InvenTree provides low-level stock control and part tracking using a Python/Django database backend with an admin interface and a REST API for integration. A powerful plugin system supports custom applications and extensions.  For more details, visit our [website](https://inventree.org).

<!-- Getting Started -->
## Getting Started with InvenTree

Choose your preferred method to deploy InvenTree:

*   [Docker](https://docs.inventree.org/en/latest/start/docker/)
*   [DigitalOcean](https://inventree.org/digitalocean)
*   [Bare Metal](https://docs.inventree.org/en/latest/start/install/)

Quick install:
```bash
wget -qO install.sh https://get.inventree.org && bash install.sh
```

Detailed installation and setup instructions can be found in the [getting started guide](https://docs.inventree.org/en/latest/start/install/).
