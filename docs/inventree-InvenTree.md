## InvenTree: The Open-Source Inventory Management Solution for Efficient Stock Control

InvenTree is a powerful, open-source inventory management system that helps you track parts and manage stock levels. Get started today by visiting the [InvenTree repository](https://github.com/inventree/InvenTree).

### Key Features:

*   **Comprehensive Stock Control:** Manage parts, track stock levels, and monitor inventory movements.
*   **Web-Based Interface:** Accessible through a user-friendly web interface for easy management.
*   **REST API:** Integrates with other applications and systems through a REST API.
*   **Plugin System:** Extends functionality with custom applications and extensions.
*   **Mobile App:**  Companion mobile app available for Android and iOS for on-the-go inventory access.
*   **Extensible Integrations:** Integrates with external applications via API, Python module, and plugins.

### Getting Started

Choose your preferred deployment method:

*   **Docker:** Deploy InvenTree with ease using Docker.
*   **Digital Ocean:** Deploy with a single click via Digital Ocean.
*   **Bare Metal:** Install directly on your server.

For detailed instructions, refer to the [installation guide](https://docs.inventree.org/en/latest/start/install/).

### Tech Stack

*   **Server:** Python, Django, DRF, Django Q, Django-Allauth
*   **Database:** PostgreSQL, MySQL, SQLite, Redis
*   **Client:** React, Lingui, React Router, TanStack Query, Zustand, Mantine, Mantine Data Table, CodeMirror
*   **DevOps:** Docker, Crowdin, Codecov, SonarCloud, Packager.io
