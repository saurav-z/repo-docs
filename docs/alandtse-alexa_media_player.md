## Key Features

This custom component empowers you to control and monitor your Amazon Echo devices directly from Home Assistant.  Here are some key features:

*   **Media Control:**
    *   Play/Pause/Stop
    *   Next/Previous Track
    *   Volume Control
*   **Media Information Display:**
    *   Song Title
    *   Artist Name
    *   Album Name
    *   Album Image
