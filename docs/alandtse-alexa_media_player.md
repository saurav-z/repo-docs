## Key Features

This custom component provides seamless integration and control over your Alexa devices, acting as media players within Home Assistant.

*   **Media Control:**
    *   Play/Pause/Stop
    *   Next/Previous Track
    *   Volume Control
*   **Media Information Retrieval:**
    *   Song Title Display
    *   Artist Name Display
    *   Album Name Display
    *   Album Image Display
