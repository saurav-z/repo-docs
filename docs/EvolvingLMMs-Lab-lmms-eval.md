# LMMs-Eval: Evaluate and Advance Large Multimodal Models

**LMMs-Eval empowers researchers and developers to efficiently evaluate and advance Large Multimodal Models (LMMs).**  This comprehensive evaluation suite supports a wide array of text, image, video, and audio tasks, providing a robust framework for assessing LMM performance.

## Key Features of LMMs-Eval:

*   **Extensive Task Coverage:** Supports a wide range of multimodal tasks, including text, image, video, and audio, ensuring comprehensive model evaluation.
*   **Broad Model Compatibility:**  Evaluates over 30 different LMMs, providing flexibility for comparing and benchmarking various models.
*   **User-Friendly Installation & Usage:**  Offers straightforward installation with `uv` (recommended) and clear examples for quick evaluation runs.
*   **Accelerated Evaluation:** Integrates with `vLLM` and supports OpenAI API compatible models, optimizing evaluation speed.
*   **Reproducibility:** Provides scripts and environment information to reproduce LLaVA-1.5 paper results and supports custom model and dataset integration.
*   **Active Development:**  Continuously updated with new tasks, models, and features, ensuring the framework remains current with the latest advancements in LMMs.

## Installation

### Using uv (Recommended)

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
git clone https://github.com/EvolvingLMMs-Lab/lmms-eval
cd lmms-eval
uv sync
```

### Alternative Installation

```bash
uv venv eval
uv venv --python 3.12
source eval/bin/activate
uv pip install git+https://github.com/EvolvingLMMs-Lab/lmms-eval.git
```

## Usages

> More examples can be found in [examples/models](examples/models)

**Evaluation of OpenAI-Compatible Model**

```bash
bash examples/models/openai_compatible.sh
bash examples/models/xai_grok.sh
```

**Evaluation of vLLM**

```bash
bash examples/models/vllm_qwen2vl.sh
```

**Evaluation of LLaVA-OneVision**

```bash
bash examples/models/llava_onevision.sh
```

**Evaluation of LLaMA-3.2-Vision**

```bash
bash examples/models/llama_vision.sh
```

**Evaluation of Qwen2-VL**

```bash
bash examples/models/qwen2_vl.sh
bash examples/models/qwen2_5_vl.sh
```

**Evaluation of LLaVA on MME**

