## ✈️ 什么是“机场”？ 科学上网的原理与作用（2025 版）

“机场”是指基于 **Shadowsocks、V2Ray、Trojan** 等加密协议构建的科学上网工具，通过**订阅链接**，用户可一键获取多国节点，突破防火墙，畅享全球网络。

相比 VPN，机场提供**多节点、智能分流**，包括：

*   ⛓️ **多种入口链路**（直连、CN2 专线等）
*   🌉 **多样过境技术**（IEPL 专线、BGP 高防网络）
*   🌍 **全球分布节点**（原生 IP、家庭宽带、云服务商 IP）

<a id="vs"></a>

## 机场 vs. VPN：哪个更适合你？

| 维度 | VPN | 机场 |
| :-- | :-- | :-- |
| **连接方式** | 官方客户端一键全局 | 订阅导入→策略组→分流 |
| **节点数量** | 3‑10 条主流出口 | 30‑200+ 全球线路 |
| **灵活度** | 全局 / 分应用（少数支持） | 域名 / 端口 / IP 分流、规则自定义 |
| **速度上限** | 受单机带宽、用户量限制 | 取决于机场总带宽投入与线路等级 |
| **价格区间** | 2‑12 USD/月 | 5‑120 CNY/月 |
| **协议可选** | OpenVPN / WireGuard 等 | SS / Vmess / Trojan 并存 |
| **解锁能力** | Netflix 仅部分；ChatGPT 看官方策略 | 针对流媒体 + AI 优化，解锁率高 |

**核心结论：**  机场更适合对速度、自由度、解锁能力有较高要求的用户；VPN 操作简单，但灵活度较低。

1.  **什么是“机场”？**  提供科学上网节点订阅的服务。

5.  **我人在中国，可以用机场吗？**  可以，推荐使用支持 Clash/V2Ray 的优质机场。
6.  **哪些平台可以用机场？**  Windows、macOS、Android、iOS、Linux。

8.  **我需要安装什么软件？**  Clash for Windows、Clash Verge、V2rayN、Shadowrocket 等。
9.  **机场速度快吗？会卡吗？**  取决于线路、带宽和晚高峰表现。
