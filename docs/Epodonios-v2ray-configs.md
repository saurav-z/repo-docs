# Free V2Ray Configurations: Access the Internet Securely & Anonymously

**Tired of internet restrictions?** This repository ([Original Repo](https://github.com/Epodonios/v2ray-configs)) provides a constantly updated collection of free V2Ray configurations, empowering you to bypass censorship and browse the web with enhanced privacy.

## Key Features:

*   **Regularly Updated:**  Thousands of V2Ray configurations are collected and refreshed every five minutes.
*   **Multiple Protocols:** Supports a wide range of protocols including Vmess, Vless, Trojan, Tuic, Shadowsocks, and ShadowsocksR.
*   **Flexible Formats:** Configurations available in base64, standard, and split formats.
*   **Cross-Platform Compatibility:** Works with popular V2Ray clients on Android, iOS, Windows, and Linux.
*   **Easy to Use:** Subscription links make it simple to stay updated with fresh configurations.

## Enhanced System Tunneling (Optional):

For comprehensive system-wide proxying, consider using a proxy program:

### Option 1: Using Proxifier

1.  **Install Proxifier:** [https://proxifier.com/download/](https://proxifier.com/download/)
2.  **Activate:** Use one of the provided activation keys.
3.  **Configure Proxy Server:** In Proxifier, go to "Profile" > "Proxy Servers" > "Add".
    *   **Address:** 127.0.0.1
    *   **Port:**
        *   V2rayN: 10808
        *   Netch: 2801
        *   SSR: 1080
        *   Mac V2rayU: 1086
    *   **Protocol:** SOCKS5
4.  **Enjoy!** Some applications might not fully tunnel; Proxifier resolves this.

### Option 2: System Proxy Configuration (without extra software):

1.  **Open System Settings:** Go to your OS's proxy settings.
2.  **Configure Proxy:**
    *   **IP:** 127.0.0.1
    *   **Port:** 10809
    *   **Local Host:**

        ```
        localhost;127.*;10.*;172.16.*;172.17.*;172.18.*;172.19.*;172.20.*;172.21.*;172.22.*;172.23.*;172.24.*;172.25.*;172.26.*;172.27.*;172.28.*;172.29.*;172.30.*;172.31.*;192.168.*
        ```
3.  **Enable Proxy:** Turn on the proxy setting.
4.  **V2Ray Client Setup:** In your V2Ray client, configure the system proxy setting.

### V2Ray Config Scanner

This section is for the Python script:

A Python script is included to scan and ping V2Ray configuration links (vmess, vless, etc.), and outputs their protocol and latency. Useful for testing and sorting multiple V2Ray configs based on performance.

## V2Ray Config Scanner Features

*   Supports vmess, vless, and other V2Ray protocols
*   Measures latency (ping) for each config
*   Sorts or filters results based on protocol and responsiveness
*   Simple, fast, and dependency-free (only requires Python)

## V2Ray Config Scanner Requirements

*   Python 3.x (no external packages required)

## V2Ray Config Scanner Usage

 1. Make sure Python 3 is installed on your system.
 2. Download the sub*.txt files from this repository (they contain lists of V2Ray subscription links).
 3. Run the script and provide the path to one or more sub*.txt files as arguments.
 4. The script will start scanning and show you the protocol and ping for each config.

Sample Output

[vmess] node1.example.com - 42 ms
[vless] node2.example.net - timeout
[shadowsocks] fastnode.org - 35 ms
