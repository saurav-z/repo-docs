## Key Features of HunyuanWorld 1.0:

*   **Immersive 360¬∞ Experiences:** Generate fully immersive 3D environments using panoramic world proxies.
*   **Mesh Export Capabilities:** Seamlessly integrate with existing computer graphics pipelines with mesh export.
*   **Interactive 3D Worlds:** Leverage disentangled object representations for rich interactivity.
*   **Text-to-3D and Image-to-3D:** Generate 3D worlds from text prompts or existing images.
*   **State-of-the-Art Performance:** Achieve superior visual quality and geometric consistency compared to other methods.

## ‚òØÔ∏è **HunyuanWorld 1.0 Overview**

HunyuanWorld 1.0 is a cutting-edge framework designed to generate immersive, explorable, and interactive 3D worlds from textual descriptions or images. It excels by combining the strengths of both video-based and 3D-based approaches, overcoming limitations in diversity, 3D consistency, and rendering efficiency.

### Architecture

The architecture of HunyuanWorld-1.0 integrates panoramic proxy generation, semantic layering, and hierarchical 3D reconstruction to create high-quality scene-scale 360¬∞ 3D worlds. It supports both text and image inputs.

## ü§ó Get Started

Follow these steps to begin using HunyuanWorld 1.0:

### Environment Setup

Install dependencies using the provided `HunyuanWorld.yaml` file and other necessary steps. Detailed instructions are available in the original repository.

```bash
git clone https://github.com/Tencent-Hunyuan/HunyuanWorld-1.0.git
cd HunyuanWorld-1.0
conda env create -f docker/HunyuanWorld.yaml
# ... other setup steps ...
```

### Code Usage

Generate 3D worlds using the provided Python scripts.

**Example: Image-to-World Generation**

```python
# Generate Panorama from Image
python3 demo_panogen.py --prompt "" --image_path examples/case2/input.png --output_path test_results/case2
# Create World Scene from Panorama
CUDA_VISIBLE_DEVICES=0 python3 demo_scenegen.py --image_path test_results/case2/panorama.png --labels_fg1 stones --labels_fg2 trees --classes outdoor --output_path test_results/case2
```

**Example: Text-to-World Generation**

```python
# Generate Panorama from Prompt
python3 demo_panogen.py --prompt "At the moment of glacier collapse, giant ice walls collapse and create waves, with no wildlife, captured in a disaster documentary" --output_path test_results/case7
# Create World Scene from Panorama
CUDA_VISIBLE_DEVICES=0 python3 demo_scenegen.py --image_path test_results/case7/panorama.png --classes outdoor --output_path test_results/case7
```

### Quick Start

Run the `scripts/test.sh` script for a quick demonstration:

```bash
bash scripts/test.sh
```
