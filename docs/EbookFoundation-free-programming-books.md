## Key Features

*   **Massive Collection:** Access thousands of free programming books, courses, cheat sheets, and more.
*   **Multi-Language Support:** Resources available in numerous languages, catering to a global audience.
*   **Comprehensive Categories:** Organized by programming language, subject, and resource type for easy navigation.
*   **Active Community:**  Benefit from community contributions and updates, ensuring a current and relevant resource.
*   **Searchable and Accessible:**  Easily find what you need with a dedicated search and user-friendly website.
*   **Open Source & Collaborative:**  Contribute to the project and help keep the resources up-to-date.

## About

This repository, a popular resource for free programming education, began as a list on StackOverflow and has since grown to be one of GitHub's most popular repositories.  It's now maintained by the [Free Ebook Foundation](https://ebookfoundation.org), a non-profit dedicated to promoting free ebooks. Find the original repo [here](https://github.com/EbookFoundation/free-programming-books).
