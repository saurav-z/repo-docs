## Key Features

*   **Extensive Collection:**  Access a comprehensive library of free programming resources.
*   **Multiple Languages:** Resources available in English and many other languages.
*   **Various Formats:** Find books, courses, cheat sheets, interactive tutorials, and more.
*   **Organized by Genre:** Browse resources categorized by programming language, subject, and format.
*   **Community Driven:**  A collaborative project, with contributions from developers worldwide.
*   **Active Development:**  Regular updates and improvements, as well as a search website.

## History

This repository, originally a clone of a StackOverflow list, has grown into a premier resource for free programming education. It is now maintained by the [Free Ebook Foundation](https://ebookfoundation.org), a non-profit organization dedicated to promoting free educational materials.
