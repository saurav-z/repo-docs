# Free Programming Books and Resources: Learn to Code for Free!

Looking to learn to code without breaking the bank? This extensive collection provides a wealth of free learning resources, including books, courses, cheat sheets, and more, in multiple languages.  Find the original repo [here](https://github.com/EbookFoundation/free-programming-books).

## Key Features

*   **Massive Resource Library:** Access a vast selection of free books, courses, cheat sheets, and interactive tutorials covering a wide range of programming languages and subjects.
*   **Multilingual Support:** Resources available in numerous languages, including English, Spanish, French, German, Chinese, and many more, catering to a global audience.
*   **Organized by Category:** Resources are neatly categorized by type (books, courses, cheat sheets, etc.) and programming language or subject, making it easy to find what you need.
*   **Community-Driven:**  This project is a collaborative effort, with contributions from developers worldwide, constantly updated to provide the most current resources.
*   **Interactive Learning:** Explore interactive programming resources, including playgrounds where you can write, compile, and run code directly in your browser.
*   **Cheat Sheets:** Quick reference guides for various languages.
*   **Podcast & Screencasts:** Free podcasts and screencasts.
*   **Translations:** Code of Conduct, Contributing, and How-to documents translated in several languages.

## Getting Started

*   **Browse the Resources:** Explore the different categories to find books, courses, and other materials that match your interests and skill level.
