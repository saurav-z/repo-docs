# Free Programming Books - Your Gateway to Open-Source Learning

**Unlock your coding potential with Free Programming Books, a comprehensive and collaborative collection of free learning resources.** Explore a vast library of books, courses, cheat sheets, and interactive tutorials, all available to help you learn to code.

## Key Features:

*   **Extensive Resource Library:** Access a massive collection of free books, courses, cheat sheets, and interactive programming tutorials.
*   **Multi-Language Support:** Learn in your preferred language with resources available in numerous languages.
*   **Community-Driven:** Contribute and collaborate with a global community of learners and educators.
*   **Regularly Updated:** Stay current with the latest learning materials, thanks to continuous community contributions.
*   **Easy to Navigate:** Browse resources by programming language, subject, and type.
*   **Open Source and Free:** All resources are freely available under the CC BY 4.0 license, ensuring accessibility.

## Getting Started

This project, originally inspired by StackOverflow, has grown into a leading repository for free programming resources. Maintained by the [Free Ebook Foundation](https://ebookfoundation.org), a non-profit dedicated to promoting free educational materials. [Donate](https://ebookfoundation.org/contributions.html) to support the cause.

*   **Contribute:** Learn how to contribute by reading the [CONTRIBUTING](docs/CONTRIBUTING.md) guide. New to GitHub? Check out the [HOWTO](docs/HOWTO.md) guide.
*   **Code of Conduct:** Ensure a positive community by adhering to our [Code of Conduct](docs/CODE_OF_CONDUCT.md).
