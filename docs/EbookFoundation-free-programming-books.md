# Free Programming Books: Your Gateway to Learning Code (and More!)

This repository is your ultimate, collaborative, and **free** resource for learning to code, featuring a massive collection of programming books, courses, and interactive tutorials in multiple languages.  ([View the original repository](https://github.com/EbookFoundation/free-programming-books))

**Key Features:**

*   **Extensive Resource Library:** Explore a vast collection of books, courses, cheat sheets, podcasts, and interactive programming resources covering various programming languages and topics.
*   **Multilingual Support:** Learn in your preferred language with resources available in English and many other languages.
*   **Community-Driven:** Contribute to the project by adding resources, improving content, and translating documentation.
*   **Dynamic Search:** Easily find the resources you need with the [dedicated search tool](https://ebookfoundation.github.io/free-programming-books-search/).
*   **Active Community:** Join a thriving community of developers and learners.
*   **Open Source and Free:** Access all resources completely free of charge.
