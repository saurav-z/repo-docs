## Key Features

*   **Extensive Collection:**  Discover a massive library of free programming books, courses, cheat sheets, interactive tutorials, and more.
*   **Multilingual Support:** Resources available in a wide array of languages, catering to learners worldwide.
*   **Organized by Category:**  Browse resources categorized by programming language, subject, and learning format.
*   **Active Community:**  Join a vibrant community contributing to and maintaining this valuable resource.
*   **Easy to Contribute:** Learn how to contribute to the project with guides for beginners.
*   **Dynamic and Static Search:** Access resources via a dynamic search or static site.

## Getting Started

Originally inspired by a StackOverflow list, this repository has grown into a comprehensive hub for free programming education, now managed by the [Free Ebook Foundation](https://ebookfoundation.org), a non-profit dedicated to promoting free ebooks.

*   **Explore the Resources:** Browse the categorized lists of books, courses, and other learning materials.
*   **Search for Specific Topics:** Use the search bar to quickly find resources related to your interests.
*   **Contribute to the Project:** Help expand the library and improve the quality of the resources by following the [CONTRIBUTING](docs/CONTRIBUTING.md) guide.

## How to Contribute

Contribute to this project by:

*   Adding new resources.
*   Improving existing entries.
*   Translating content.
*   Fixing issues.

Read [CONTRIBUTING](docs/CONTRIBUTING.md) and [HOWTO](docs/HOWTO.md) for contribution guidelines.  Adhere to the [Code of Conduct](docs/CODE_OF_CONDUCT.md).
