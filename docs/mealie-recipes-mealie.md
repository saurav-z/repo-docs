## About Mealie

**Mealie** is a self-hosted recipe manager designed to simplify your cooking life. With a user-friendly interface built with Vue and a REST API backend, Mealie makes it easy to add, organize, and access your favorite recipes. Import recipes from URLs, manually add family recipes, plan your meals, and generate shopping lists – all in one convenient platform. This project is available on [GitHub](https://github.com/mealie-recipes/mealie).

### Key Features:

*   **Recipe Import:** Easily import recipes from any URL, or manually add your own recipes.
*   **Meal Planner:** Plan your weekly meals with the built-in meal planner.
*   **Shopping List:** Generate shopping lists, organized by supermarket sections.
*   **Cookbooks:** Organize recipes into custom cookbooks.
*   **Docker Support:** Simple deployment using Docker.
*   **Localization:** Available in 35+ languages.
