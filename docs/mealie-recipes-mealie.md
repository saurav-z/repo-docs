## Mealie: Your Self-Hosted Recipe Manager and Meal Planner

Mealie is a feature-rich, open-source recipe manager and meal planner that simplifies cooking and meal organization for you and your family. ([See the original repo](https://github.com/mealie-recipes/mealie))

[![Product Name Screen Shot][product-screenshot]](https://docs.mealie.io)

### Key Features:

*   **Recipe Import:** Effortlessly add recipes by importing from URLs or manually entering details.
*   **Meal Planner:** Plan your weekly meals with an easy-to-use meal planner.
*   **Shopping List:** Generate organized shopping lists, categorized by supermarket sections.
*   **Cookbooks:** Organize recipes into custom cookbooks for easy access.
*   **Docker Support:** Deploy Mealie quickly and easily with Docker.
*   **Localization:** Available in 35+ languages.

### About the Project

Mealie provides a user-friendly experience with a reactive frontend built in Vue and a REST API backend. It allows you to easily add recipes, plan your meals, and generate shopping lists.
