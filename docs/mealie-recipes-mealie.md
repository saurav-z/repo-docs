## Mealie: Your Self-Hosted Recipe Manager and Meal Planner

Mealie is the ultimate open-source solution for managing your recipes, planning meals, and creating shopping lists, all in one user-friendly platform.  Find the original repository [here](https://github.com/mealie-recipes/mealie).

[![Product Name Screen Shot][product-screenshot]](https://docs.mealie.io)

## Key Features

*   **Recipe Import:** Easily add recipes by importing them from a URL or manually entering the details.
*   **Meal Planner:** Plan your weekly meals with an intuitive meal planning tool.
*   **Shopping List:** Generate organized shopping lists, sorted by supermarket sections.
*   **Cookbooks:** Organize your recipes into custom cookbooks for easy access.
*   **Docker Support:**  Deploy and manage Mealie with ease using Docker.
*   **Multi-Language Support:**  Available in 35+ languages to cater to a global audience.

## About the Project

Mealie is a self-hosted recipe manager, meal planner, and shopping list application. With a focus on user experience, Mealie boasts a reactive frontend built with Vue and a REST API backend. You can effortlessly add recipes by importing them from a URL, or manually creating your own using the UI editor.  Mealie also offers an API, enabling integration with third-party applications.
