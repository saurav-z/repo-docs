SurfSense is a highly customizable AI research agent that integrates with your personal knowledge base and external sources to streamline your research.

## Key Features

*   üí° **Private Knowledge Base Integration:** Combine the power of tools like NotebookLM and Perplexity with your personal data.
*   üìÅ **Extensive File Format Support:** Upload and save content from various file formats to your personal knowledge base.
*   üîç **Powerful Search:** Quickly find information within your saved content using advanced search capabilities.
*   üí¨ **Interactive Chat:** Engage in natural language conversations with your saved content and receive cited answers.
*   üìÑ **Cited Answers:** Get trustworthy answers with citations, similar to Perplexity.
*   üîî **Privacy & Local LLM Support:** Works seamlessly with local LLMs like Ollama for enhanced privacy.
*   üè† **Self-Hosted & Open Source:** Deploy SurfSense locally with ease.
*   üéôÔ∏è **Automated Podcast Generation:**
    *   Blazingly fast podcast generation (3-minute podcasts in under 20 seconds).
    *   Convert chat conversations into audio content.
    *   Support for local and multiple TTS providers (Kokoro TTS, OpenAI, Azure, Google Vertex AI).
*   üìä **Advanced RAG Techniques:**
    *   Supports 100+ LLMs.
    *   Supports 6000+ Embedding Models.
    *   Supports major Rerankers (Pinecone, Cohere, Flashrank, etc.).
    *   Uses Hierarchical Indices (2-tiered RAG setup).
    *   Hybrid Search (Semantic + Full Text Search with Reciprocal Rank Fusion).
    *   RAG as a Service API Backend.
*   ‚ÑπÔ∏è **External Source Integration:** Connect to various sources, including:
    *   Search Engines (Tavily, LinkUp)
    *   Slack, Linear, Jira, ClickUp, Confluence, Notion, Gmail, YouTube, GitHub, Discord, Google Calendar, and more.
*   üîñ **Cross-Browser Extension:** Save webpages easily with a browser extension, including those behind authentication.
