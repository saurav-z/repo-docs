*   **Quick Install (Mac/Linux):**
    ```bash

    ```
*   **Quick Install (Windows/Cross-platform):**
    ```bash
    python install.py
    ```
*   **Manual Install:**
    ```bash
    git clone https://github.com/brennercruvinel/CCPlugins.git
    cd CCPlugins
    python install.py
    ```
