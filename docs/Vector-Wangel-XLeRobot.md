# XLeRobot: Build Your Own Affordable Household Robot 🤖

**XLeRobot empowers everyone to build and experiment with advanced embodied AI, offering a cost-effective, open-source robotics platform that's more affordable than a new phone!** [Check out the original repository](https://github.com/Vector-Wangel/XLeRobot).

## Key Features

*   **Affordable:** Starting at approximately \$660 USD, XLeRobot makes advanced robotics accessible.
*   **Fast Assembly:** Build your robot in under 4 hours.
*   **Open Source:**  Built upon the shoulders of giants like [LeRobot](https://github.com/huggingface/lerobot), [SO-100/SO-101](https://github.com/TheRobotStudio/SO-ARM100), [Lekiwi](https://github.com/SIGRobotics-UIUC/LeKiwi), and [Bambot](https://github.com/timqian/bambot), fostering community and collaboration.
*   **Real-World Control:** Control XLeRobot in real-time with an Xbox controller, even without Wi-Fi, with zero latency.
*   **VR Integration:** XLeVR provides full-body control via VR Quest 3, with minimal dependencies and a quick setup time.
*   **Official Simulation:**  Includes updated URDFs and control scripts, making it easy to simulate and test your robot.
*   **Comprehensive Documentation:** [Explore detailed tutorials, demos, and resources](https://xlerobot.readthedocs.io/en/latest/index.html) on the project's documentation website.

## Getting Started

> [!NOTE]
> This guide is designed to be beginner-friendly.

> [!NOTE]
> If you're new to programming, familiarize yourself with Python, Ubuntu, and Git before starting.

1.  💵 **Buy your parts**: [Bill of Materials](https://xlerobot.readthedocs.io/en/latest/hardware/getting_started/material.html)
