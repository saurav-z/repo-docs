## Key Features

*   **Unified API:** Call all LLM APIs (OpenAI, Azure, Bedrock, Hugging Face, VertexAI, TogetherAI, Groq, etc.) using the familiar OpenAI format.
*   **Consistent Output:**  Get consistent results for text responses, always available at `['choices'][0]['message']['content']`.
*   **Intelligent Routing:** Implement retry and fallback logic across multiple deployments (e.g., Azure/OpenAI) with the [Router](https://docs.litellm.ai/docs/routing).
*   **Cost Control & Management**: Set budgets and rate limits per project, API key, and model using the [LiteLLM Proxy Server (LLM Gateway)](https://docs.litellm.ai/docs/simple_proxy).
*   **Asynchronous Support:** Benefit from efficient processing with async completion and streaming capabilities.
*   **Streaming Support:** Stream model responses for faster user experiences.

## Quickstart

```bash
pip install litellm
```

```python
from litellm import completion
import os

## set ENV variables
os.environ["OPENAI_API_KEY"] = "your-openai-key"
os.environ["ANTHROPIC_API_KEY"] = "your-anthropic-key"

messages = [{ "content": "Hello, how are you?","role": "user"}]

# openai call
response = completion(model="openai/gpt-4o", messages=messages)

# anthropic call
response = completion(model="anthropic/claude-sonnet-4-20250514", messages=messages)
print(response)
```

## LiteLLM Proxy Server (LLM Gateway)

Manage costs, track spending, and load balance across multiple projects with the LiteLLM Proxy.

*   **Authentication:** Hooks for custom authentication.
*   **Logging:** Custom logging capabilities.
*   **Cost Tracking:** Monitor LLM spend.
*   **Rate Limiting:** Implement rate limits per user or project.

**Quick Start Proxy (CLI)**

```bash
pip install 'litellm[proxy]'
```

1.  **Start the proxy:**

    ```bash
    litellm --model huggingface/bigcode/starcoder
    #INFO: Proxy running on http://0.0.0.0:4000
    ```

2.  **Make a ChatCompletions request (using the OpenAI SDK):**

    ```python
    import openai # openai v1.0.0+
    client = openai.OpenAI(api_key="anything",base_url="http://0.0.0.0:4000") # set proxy to base_url
    # request sent to model set on litellm proxy, `litellm --model`
    response = client.chat.completions.create(model="gpt-3.5-turbo", messages = [
        {
            "role": "user",
            "content": "this is a test request, write a short poem"
        }
    ])

    print(response)
    ```

**Proxy Key Management:**

Create proxy keys with a Postgres DB. See the [Proxy Key Management Docs](https://docs.litellm.ai/docs/proxy/virtual_keys) for details.

*   UI available at `/ui` on your proxy server.
*   Set budgets and rate limits via `POST /key/generate`.
