## Key Features

*   **Unified API:** Interact with diverse LLM providers (OpenAI, Azure, Anthropic, Cohere, etc.) using a consistent OpenAI-compatible format.
*   **Consistent Output:**  Receive predictable text responses always available at `['choices'][0]['message']['content']`.
*   **Intelligent Routing:** Built-in retry and fallback logic across multiple deployments (e.g., Azure/OpenAI).
*   **Cost Management & Rate Limiting:**  Control budgets and rate limits per project, API key, and model using the LiteLLM Proxy Server.
*   **Async Support:** Utilize asynchronous functions for non-blocking operations, improving performance.
*   **Streaming Support:** Get real-time responses with streaming capabilities, supported across all models.
*   **Observability & Logging:**  Integrate with popular logging platforms (Lunary, MLflow, Langfuse, etc.) for comprehensive monitoring.

## Quickstart

Install LiteLLM:

```bash
pip install litellm
```

Example usage:

```python
from litellm import completion
import os

# Set your API keys
os.environ["OPENAI_API_KEY"] = "your-openai-key"
os.environ["ANTHROPIC_API_KEY"] = "your-anthropic-key"

messages = [{"content": "Hello, how are you?","role": "user"}]

# OpenAI call
response = completion(model="openai/gpt-4o", messages=messages)

# Anthropic call
response = completion(model="anthropic/claude-sonnet-4-20250514", messages=messages)
print(response)
```

##  Asynchronous Operations ([Docs](https://docs.litellm.ai/docs/completion/stream#async-completion))

```python
from litellm import acompletion
import asyncio

async def test_get_response():
    user_message = "Hello, how are you?"
    messages = [{"content": user_message, "role": "user"}]
    response = await acompletion(model="openai/gpt-4o", messages=messages)
    return response

response = asyncio.run(test_get_response())
print(response)
```

## Streaming Support ([Docs](https://docs.litellm.ai/docs/completion/stream))

```python
from litellm import completion
response = completion(model="openai/gpt-4o", messages=messages, stream=True)
for part in response:
    print(part.choices[0].delta.content or "")

# claude sonnet 4
response = completion('anthropic/claude-sonnet-4-20250514', messages, stream=True)
for part in response:
    print(part)
```

## Logging & Observability ([Docs](https://docs.litellm.ai/docs/observability/callbacks))

Integrate with various logging services for monitoring:

```python
from litellm import completion
import os

# Set up logging environments
os.environ["LUNARY_PUBLIC_KEY"] = "your-lunary-public-key"
os.environ["HELICONE_API_KEY"] = "your-helicone-auth-key"
os.environ["LANGFUSE_PUBLIC_KEY"] = ""
os.environ["LANGFUSE_SECRET_KEY"] = ""
os.environ["ATHINA_API_KEY"] = "your-athina-api-key"

os.environ["OPENAI_API_KEY"] = "your-openai-key"

# Set callbacks
litellm.success_callback = ["lunary", "mlflow", "langfuse", "athina", "helicone"]

# Call OpenAI
response = completion(model="openai/gpt-4o", messages=[{"role": "user", "content": "Hi ðŸ‘‹ - i'm openai"}])
```
