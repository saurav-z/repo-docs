## What is Klavis AI?

Klavis AI provides robust MCP (Message Control Protocol) integration layers, enabling AI agents to reliably utilize thousands of tools. This allows you to connect your AI to the apps you use every day with enterprise OAuth and Docker support.

**Key Features:**

*   **Scalable Tool Integration:** Go beyond the limits of typical tool integrations.
*   **Progressive Discovery:** Guides AI agents through complex workflows step-by-step.
*   **50+ Production MCP Servers:** Connect to popular services like GitHub, Gmail, and Slack.
*   **Enterprise-Grade OAuth:** Secure and reliable authentication.
*   **Docker Ready:** Easy deployment and scaling.
*   **SDK Support:** Python and TypeScript SDKs available.
*   **REST API:** Integrate with any programming language.

## üöÄ Quick Start Guide

Get started with Klavis AI in minutes:

### Option 1: Self-Hosting with Open Source

1.  **Run any MCP Integration:**
    ```bash
    docker pull ghcr.io/klavis-ai/github-mcp-server:latest
    docker run -p 5000:5000 ghcr.io/klavis-ai/github-mcp-server:latest
    ```

2.  **Install Open Source Strata locally:**
    ```bash
    pipx install strata-mcp
    strata add --type stdio playwright npx @playwright/mcp@latest
    ```

### Option 2: Use the Hosted Service by WebUI

1.  **Sign Up:** [Create an Account ‚Üí](https://www.klavis.ai/auth/sign-up)
2.  **Get Started:** [Follow the Quickstart Guide ‚Üí](https://docs.klavis.ai/documentation/quickstart)

### Option 3: SDK Integration

#### Python

```python
from klavis import Klavis
from klavis.types import McpServerName

klavis = Klavis(api_key="your-key")

# Create Strata instance
strata = klavis.mcp_server.create_strata_server(
    user_id="user123",
    servers=[McpServerName.GMAIL, McpServerName.YOUTUBE],
)

# Or use individual MCP servers
gmail = klavis.mcp_server.create_server_instance(
    server_name=McpServerName.GMAIL,
    user_id="user123",
)
```

#### TypeScript

```typescript
import { KlavisClient, McpServerName } from 'klavis';

const klavis = new KlavisClient({ apiKey: 'your-api-key' });

// Create Strata instance
const strata = await klavis.mcpServer.createStrataServer({
    userId: "user123",
    servers: [McpServerName.GMAIL, McpServerName.YOUTUBE]
});

// Or use individual MCP servers
const gmail = await klavis.mcpServer.createServerInstance({
    serverName: McpServerName.GMAIL,
    userId: "user123"
});
```

### Option 4: Direct API Integration

#### Create Strata Server

```bash
curl -X POST "https://api.klavis.ai/v1/mcp-server/strata" \
  -H "Authorization: Bearer your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "user123",
    "servers": ["GMAIL", "YOUTUBE"]
  }'
```

#### Create Individual MCP Server

```bash
curl -X POST "https://api.klavis.ai/v1/mcp-server/instance" \
  -H "Authorization: Bearer your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "server_name": "GMAIL",
    "user_id": "user123"
  }'
```
