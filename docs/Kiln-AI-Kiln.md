## Install Python Library

[![PyPI - Version](https://img.shields.io/pypi/v/kiln-ai.svg?logo=pypi&label=PyPI&logoColor=gold)](https://pypi.org/project/kiln-ai/) [![Docs](https://img.shields.io/badge/docs-pdoc-blue)](https://kiln-ai.github.io/Kiln/kiln_core_docs/index.html)

Integrate Kiln datasets seamlessly into your AI workflows with our open-source [python library](https://pypi.org/project/kiln-ai/). Fine-tune models, and create custom tools to expand your AI capabilities.  [Read the docs](https://kiln-ai.github.io/Kiln/kiln_core_docs/index.html) to get started.

```bash
pip install kiln-ai
```

## Rapid AI Prototyping and Dataset Collaboration

*   **Accelerated Prototyping:** Quickly experiment with different prompting techniques, models (GPT, Llama, Claude, etc.), and fine-tuning options within Kiln.  Iterate efficiently to optimize AI quality and performance.
*   **Cross-Team Collaboration:** Facilitate seamless collaboration between technical and non-technical teams.  Subject matter experts can create structured datasets, while data scientists can leverage them using the UI or the Python library.  Improve team efficiency through the use of a git-based dataset file format.
