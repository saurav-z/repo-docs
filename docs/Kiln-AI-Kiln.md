## About Kiln

Kiln is the collaborative AI platform that streamlines AI development, making it easier to prototype, fine-tune, and deploy AI models with seamless dataset collaboration.  Explore the [Kiln repository](https://github.com/Kiln-AI/Kiln) to learn more.

## Key Features

*   **Intuitive Desktop Apps:** One-click apps for Windows, MacOS, and Linux, with a user-friendly design.
*   **Zero-Code Fine-Tuning:** Fine-tune models like Llama, GPT-4o, and more without writing any code, and deploy them serverless.
*   **AI Model Evaluation:** Assess your model quality with state-of-the-art evaluators.
*   **Synthetic Data Generation:** Generate training data using our interactive visual tools.
*   **Custom Reasoning Models:** Train and refine your own custom reasoning models.
*   **Git-Based Collaboration:** Version control for your AI datasets, allowing easy collaboration with QA, PM, and subject matter experts.
*   **Automated Prompt Generation:** Generate prompts automatically for various techniques like chain-of-thought, few-shot, and more.
*   **Wide Model & Provider Support:** Works with models from Ollama, OpenAI, OpenRouter, Fireworks, Groq, AWS, and any OpenAI-compatible API.
*   **Open Source and API:** Utilizes an open-source Python library and OpenAPI REST API (MIT License).
*   **Privacy-First:** Offers data privacy by allowing you to bring your own API keys or run models locally with Ollama.
*   **Structured Data:** Build AI tasks with JSON data structures.
*   **Free to Use:** Offers free apps and an open-source library.

## Learn More About Kiln

### Rapid Prototyping

Kiln simplifies experimentation with cutting-edge models and techniques, enabling rapid comparison without code.

Key features include:

*   Prompting: Basic, few-shot, multi-shot, repair, and feedback techniques
*   Chain-of-thought and custom “thinking” instructions
*   Model Support: GPT, Llama, Claude, Gemini, Mistral, Gemma, Phi, and more
*   Fine-Tuning: Create custom models using your Kiln datasets

Future plans include advanced features such as evals and RAG.  Experienced users can start using these techniques with Kiln datasets and the Python library today.

### Collaboration Across Teams

Kiln bridges the gap between subject matter experts and technical teams in AI product development.

*   Subject matter experts create structured datasets and ratings via intuitive desktop apps.
*   Data scientists utilize the UI or delve deeper with the Python library.
*   QA and PM identify and resolve issues by contributing to the dataset.

Kiln utilizes a dataset file format compatible with Git, for collaborative editing and attribution. Even non-technical team members can contribute data and evals without needing to know Git.

### Build High-Quality AI Products with Datasets

Kiln helps you build datasets for your product by capturing inputs, outputs, ratings, and feedback.

*   Model quality improves automatically as the dataset grows.
*   Iterate the dataset to address product changes and issues.
