## Key Benefits

Kiln accelerates your AI workflow by:

*   **Rapid Prototyping:** Quickly experiment with various models and techniques without writing code, leading to improved performance.
*   **Seamless Collaboration:** Connect subject matter experts with technical teams for efficient dataset creation and model development.
*   **Building High-Quality AI Products:** Improve model quality by continuously refining your dataset with inputs, outputs, ratings, and feedback.

## Install Python Library

[![PyPI - Version](https://img.shields.io/pypi/v/kiln-ai.svg?logo=pypi&label=PyPI&logoColor=gold)](https://pypi.org/project/kiln-ai/) [![Docs](https://img.shields.io/badge/docs-pdoc-blue)](https://kiln-ai.github.io/Kiln/kiln_core_docs/index.html)

Integrate Kiln datasets seamlessly into your workflows with our open-source [Python library](https://pypi.org/project/kiln-ai/). Build fine-tunes, use Kiln in notebooks, create custom tools, and more!

```bash
pip install kiln-ai
```
