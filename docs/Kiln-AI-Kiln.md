## Why Choose Kiln?

### Accelerate Prototyping

Kiln simplifies the process of experimenting with new models and techniques, allowing you to compare different approaches with ease, all without the need to write code.

Key features include:

-   Support for various prompting techniques (basic, few-shot, multi-shot, repair & feedback)
-   Chain of thought / thinking, with customizable instructions
-   Compatibility with numerous models: GPT, Llama, Claude, Gemini, Mistral, Gemma, Phi
-   Fine-tuning capabilities for custom models with Kiln datasets
-   Evaluation using LLM-as-Judge and G-Eval
-   Model distillation

We plan to add more no-code features like RAG. Experienced data scientists can currently implement such techniques using Kiln datasets and our Python library.

### Foster Collaboration

Kiln bridges the gap between subject matter experts and technical teams.

Subject matter experts can easily generate structured datasets and ratings using our intuitive desktop apps.

Data scientists can utilize the datasets, either through the UI or our Python library.

QA and PM can identify issues early and support dataset creation for model fixes.

The dataset file format supports Git for robust collaboration. Avoid collisions with UUIDs. Contribute data and evals without requiring Git.

### Drive High-Quality AI Product Development with Datasets

Kiln empowers you to create datasets to improve your AI products. We capture inputs, outputs, human ratings, and feedback to build high-quality models. Synthetic data generation enables you to build datasets for evals and fine-tuning rapidly.
Iterate your dataset as your goals shift and new bugs surface.
