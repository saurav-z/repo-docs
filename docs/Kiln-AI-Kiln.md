## About Kiln

**Kiln is an all-in-one platform that streamlines AI development by offering rapid prototyping tools, enabling seamless dataset collaboration, and facilitating the creation of high-quality AI products.**  Kiln empowers developers and teams to efficiently build, refine, and deploy AI models.

## Key Features

*   **Intuitive Desktop Apps:** Get started quickly with user-friendly desktop applications for Windows, macOS, and Linux.
*   **Fine-tuning:** Easily fine-tune leading LLMs, including Llama and GPT-4o, with zero-code options and automatic deployment.
*   **Advanced Evaluations:** Utilize state-of-the-art evaluators to accurately assess and enhance your model's performance.
*   **Synthetic Data Generation:** Generate training data with an intuitive visual tooling to expedite your project.
*   **Reasoning Models:** Design, train, and customize reasoning models tailored to your specific needs.
*   **Collaborative Workflows:** Leverage Git-based version control for AI datasets, simplifying collaboration with QA, PM, and subject matter experts through an intuitive UI.
*   **Automated Prompt Generation:** Automatically create effective prompts from your data, using techniques like chain-of-thought, few-shot, and multi-shot, and more.
*   **Broad Model & Provider Compatibility:** Integrate with diverse models via Ollama, OpenAI, OpenRouter, Fireworks, Groq, AWS, or any OpenAI-compatible API.
*   **Open-Source and API Access:** Take advantage of an open-source Python library and REST API, both licensed under MIT.
*   **Privacy-First Design:** Protect your data with a privacy-focused approach. Use your own API keys or run models locally with Ollama.
*   **Structured Data Support:** Build AI tasks that leverage JSON-based data structures.
*   **Completely Free:** Benefit from free desktop apps and an open-source library.

## Learn More

### Rapid AI Prototyping

Kiln accelerates experimentation and model comparisons.

Kiln Currently Supports:

*   Prompting Techniques: Basic, few-shot, multi-shot, repair & feedback
*   Chain of thought / thinking, with optional custom “thinking” instructions
*   Models: GPT, Llama, Claude, Gemini, Mistral, Gemma, Phi
*   Fine Tuning: create custom models using your Kiln dataset

### Team Collaboration

Kiln facilitates collaboration between technical and non-technical teams.

### Build High Quality AI Products with Datasets

Kiln helps you build high-quality models by capturing inputs, outputs, human ratings, feedback, and corrections.
