## Key Features

*   **Intuitive Desktop Apps:** Get started quickly with one-click apps for Windows, MacOS, and Linux.
*   **Zero-Code Fine-Tuning:** Fine-tune LLMs like Llama, GPT-4o, and more without writing any code.  Benefit from automatic serverless model deployment.
*   **Advanced Evaluations:** Evaluate the performance of your models and tasks using cutting-edge evaluators.
*   **Interactive Synthetic Data Generation:** Create high-quality training data using our intuitive visual tools.
*   **Custom Reasoning Models:** Train or refine custom reasoning models to meet your specific needs.
*   **Seamless Team Collaboration:** Leverage Git-based version control for your AI datasets, improving collaboration with QA, PMs, and subject matter experts on structured data.
*   **Automated Prompt Generation:** Generate prompts automatically from your data with advanced techniques including chain-of-thought and few-shot learning.
*   **Broad Model and Provider Support:** Use a variety of models through Ollama, OpenAI, OpenRouter, Fireworks, Groq, AWS, or any compatible API.
*   **Open-Source Library and API:** Take advantage of our open-source Python library and OpenAPI REST API, both MIT licensed.
*   **Privacy-Focused Design:** Protect your data with our privacy-first approach: use your own API keys or run locally with Ollama.
*   **Structured Data Support:** Build AI tasks that utilize JSON for structured data.
*   **Completely Free:** Access all features without charge; both the apps and the library are open-source.

## Learn More

### Rapid Prototyping

Kiln empowers you to swiftly explore and compare various AI techniques and models without writing any code. This results in improvements in quality and performance of your AI product.

Current features include:

*   Various prompting techniques: basic, few-shot, multi-shot, repair & feedback
*   Chain of thought / thinking, with optional custom “thinking” instructions
*   Support for a wide range of models: GPT, Llama, Claude, Gemini, Mistral, Gemma, Phi
*   Fine Tuning: Build custom models using your Kiln dataset

Future plans include no-code evals and RAG. For data scientists, build these today with Kiln datasets and our python library.

### Collaborative AI Development

Kiln bridges the gap between subject matter experts and technical teams when building AI products.

Subject matter experts can use our intuitive desktop apps to create structured datasets and ratings, eliminating the need for coding.

Data scientists can utilize the UI or the python library to work with the datasets created by SMEs.

QA and PM teams can easily identify issues and collaborate on dataset creation, which helps to improve the model.

The dataset file format allows Git for effective collaboration. UUIDs prevent collisions, while attribution is recorded within dataset files.

### Build High-Quality AI Products with Datasets

Kiln enables the creation of high-quality models for your product. Capture inputs, outputs, human ratings, feedback, and necessary repairs every time you use Kiln. The more you use it, the more data you'll have.

Your model quality will automatically improve as the dataset grows.

As product goals or bugs evolve, you can easily iterate your dataset to address issues.
