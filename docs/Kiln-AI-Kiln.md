## Learn More About Kiln

### Rapid Prototyping

Kiln simplifies experimenting with new AI models and techniques, allowing you to compare approaches effortlessly without coding.

Key features include:

*   Diverse prompting techniques: basic, few-shot, multi-shot, repair & feedback.
*   Chain-of-thought functionality, with customizable "thinking" instructions.
*   Wide model support: GPT, Llama, Claude, Gemini, Mistral, Gemma, Phi, and many more.
*   Fine-tuning: Develop custom models using your Kiln datasets.

We plan to add more advanced features like evals and RAG.  Experienced users can implement these today with Kiln datasets and our Python library.

### Collaboration Across Teams

Kiln bridges the gap between technical and non-technical teams when building AI products.

Subject matter experts can generate structured datasets and ratings without needing to code or use technical tools.

Data scientists can utilize the UI or Python library for dataset analysis.

QA and PM can easily identify and address issues.

The dataset file format is designed for Git-based version control.

### Build High-Quality AI Products with Datasets

Kiln helps you build robust datasets, capturing inputs, outputs, human ratings, feedback, and repairs. The more you use Kiln, the better your data and the better your models get.

Easily iterate the dataset to improve model performance and address issues.
