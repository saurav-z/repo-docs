# LanceDB: The Open Source Multimodal AI Lakehouse

**LanceDB** is a high-performance, open-source vector database designed for building and deploying AI applications with ease, offering a powerful and scalable solution for storing, indexing, and searching multimodal data. **[Explore LanceDB on GitHub](https://github.com/lancedb/lancedb)**

**Key Features:**

*   **Fast Vector Search:** Quickly search billions of vectors in milliseconds with state-of-the-art indexing.
*   **Multimodal Data Support:** Store and query diverse data types, including vectors, text, images, videos, and more.
*   **Comprehensive Search Capabilities:** Leverage vector similarity search, full-text search, and SQL queries for flexible data exploration.
*   **Open Source & Flexible:**  Use LanceDB locally or in the cloud, with no vendor lock-in.
*   **Advanced Features:**  Benefit from zero-copy data access, automatic versioning, and GPU support for efficient indexing.
*   **Seamless Integrations:**  Integrate easily with Python, Node.js, Rust, and REST APIs, plus popular tools like LangChain and LlamaIndex.
*   **Columnar Storage:**  Built on the Lance columnar format for efficient storage and analytics.

## How to Get Started

1.  **Installation:** Follow the [Quickstart guide](https://lancedb.github.io/lancedb/basic/) to set up LanceDB locally.

2.  **SDKs & APIs:**  Access LanceDB via multiple SDKs and a REST API:

    | Interface      | Documentation                                                    |
    | -------------- | ---------------------------------------------------------------- |
    | Python SDK     | [https://lancedb.github.io/lancedb/python/python/](https://lancedb.github.io/lancedb/python/python/)     |
    | Typescript SDK | [https://lancedb.github.io/lancedb/js/globals/](https://lancedb.github.io/lancedb/js/globals/)     |
    | Rust SDK       | [https://docs.rs/lancedb/latest/lancedb/index.html](https://docs.rs/lancedb/latest/lancedb/index.html) |
    | REST API       | [https://docs.lancedb.com/api-reference/introduction](https://docs.lancedb.com/api-reference/introduction)       |
