## **Why Choose LanceDB?**

<img max-width="750px" alt="LanceDB Multimodal Search" src="https://github.com/lancedb/lancedb/assets/917119/09c5afc5-7816-4687-bae4-f2ca194426ec">

*   **Supercharge your AI applications** with vector search, full-text search and SQL
*   **Explore the open-source AI lakehouse** to store, index, query and filter all your data.

## **Key Features of LanceDB:**

*   **Blazing-Fast Vector Search:** Search billions of vectors in milliseconds using state-of-the-art indexing.
*   **Comprehensive Search Capabilities:** Execute vector similarity search, full-text search, and SQL queries all in one place.
*   **Multimodal Data Support:** Store, query, and filter vectors, metadata, and a variety of multimodal data types (text, images, videos, point clouds, and more).
*   **Advanced Features:** Enjoy zero-copy data access, automatic versioning for data management, and GPU support for efficient vector index building.
*   **Open Source & Local**: 100% open source, runs locally or in your cloud. No vendor lock-in.
*   **Cloud and Enterprise**: Production-scale vector search with no servers to manage. Complete data sovereignty and security.
*   **Columnar Storage**: Built on the Lance columnar format for efficient storage and analytics.
*   **Seamless Integration**: Python, Node.js, Rust, and REST APIs for easy integration. Native Python and Javascript/Typescript support.
*   **Rich Ecosystem**: Integrations with [**LangChain** ü¶úÔ∏èüîó](https://python.langchain.com/docs/integrations/vectorstores/lancedb/), [**LlamaIndex** ü¶ô](https://gpt-index.readthedocs.io/en/latest/examples/vector_stores/LanceDBIndexDemo.html), Apache-Arrow, Pandas, Polars, DuckDB and more on the way.

## **How to Install LanceDB**

Getting started with LanceDB is easy! Follow these steps:

1.  **Quickstart:** Follow the [Quickstart](https://lancedb.github.io/lancedb/basic/) documentation to set up LanceDB locally.
2.  **SDKs:** We support Python, Typescript and Rust SDKs
