# LanceDB: The Open-Source, High-Performance Vector Database for AI

**LanceDB** is the ultimate open-source, multimodal AI lakehouse, designed for blazing-fast vector search and efficient storage of your AI/ML data.  ([See the original repo](https://github.com/lancedb/lancedb))

## Key Features of LanceDB:

*   **Blazing-Fast Vector Search**:  Quickly search billions of vectors with state-of-the-art indexing, delivering results in milliseconds.
*   **Comprehensive Search Capabilities**: Supports a variety of search methods, including vector similarity search, full-text search, and SQL queries.
*   **Multimodal Data Support**:  Seamlessly store, query, and filter vectors, metadata, and a wide range of multimodal data types (text, images, videos, point clouds, and more).
*   **Advanced Features for Production**: Leverage zero-copy data access, automatic versioning for data management, and GPU support for faster index building.

## Why Choose LanceDB?

*   **Open Source & Local**: LanceDB is 100% open source, allowing you to run it locally or in your cloud environment, avoiding vendor lock-in.
*   **Cloud & Enterprise Solutions**: Offers production-scale vector search capabilities, providing complete data sovereignty and security without server management overhead.
*   **Efficient Columnar Storage**: Built on the Lance columnar format, optimizing storage and analytical performance.
*   **Seamless Integration**: Offers Python, Node.js, Rust, and REST APIs for easy integration into your projects.  Native Python and Javascript/Typescript support.
*   **Extensive Ecosystem**:  Integrates with popular tools like [LangChain](https://python.langchain.com/docs/integrations/vectorstores/lancedb/), [LlamaIndex](https://gpt-index.readthedocs.io/en/latest/examples/vector_stores/LanceDBIndexDemo.html), Apache Arrow, Pandas, Polars, DuckDB, and more.
