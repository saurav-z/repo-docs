# LanceDB: The Open Source Vector Database for AI

**LanceDB** is an open-source vector database designed for storing, indexing, and searching multimodal data, enabling you to build and deploy AI applications with ease.  [**Explore the LanceDB repository on GitHub**](https://github.com/lancedb/lancedb).

## Key Features:

*   **Blazing-Fast Vector Search:** Quickly search billions of vectors with state-of-the-art indexing for sub-millisecond performance.
*   **Multimodal Data Support:** Store, query, and filter various data types, including vectors, text, images, videos, and more, all in one place.
*   **Comprehensive Search Capabilities:** Leverage vector similarity search, full-text search, and SQL for flexible data exploration.
*   **Open Source & Flexible:** 100% open source, offering complete control and no vendor lock-in, deployable locally or in your cloud environment.
*   **Production-Ready Features:** Benefit from zero-copy data access, automatic versioning, and GPU acceleration for index building.
*   **Seamless Integration:** Native Python and Javascript/Typescript support, with SDKs for Python, Typescript, and Rust. Integration with LangChain, LlamaIndex, Apache-Arrow, Pandas, Polars, DuckDB and many more.
*   **Cloud and Enterprise Options**: Production-scale vector search with no servers to manage. Complete data sovereignty and security.

##  Why Choose LanceDB?

LanceDB is the ultimate multimodal data platform for AI/ML applications, providing a central location for developers to build, train, and analyze their AI workloads. Built on the Lance columnar format, LanceDB offers efficient storage and analytics capabilities.
