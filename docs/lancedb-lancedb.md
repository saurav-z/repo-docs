# LanceDB: The Open-Source Multimodal AI Lakehouse for Vector Search and More

[**How to Install** ](#how-to-install) ‚ú¶ [**Detailed Documentation**](https://lancedb.github.io/lancedb/) ‚ú¶ [**Tutorials and Recipes**](https://github.com/lancedb/vectordb-recipes/tree/main) ‚ú¶  [**Contributors**](#contributors) 

**LanceDB is a fast, scalable, and versatile vector database designed to power the next generation of AI and ML applications.**

LanceDB provides a central, high-performance platform for developers to build, train, and analyze AI workloads with ease, offering efficient storage, indexing, and search capabilities for petabytes of multimodal data and vectors.

</div>

<br>

## **Why Choose LanceDB?**

*   **Blazing-Fast Vector Search**: Experience millisecond search times across billions of vectors using state-of-the-art indexing.
*   **Comprehensive Search Capabilities**: Perform vector similarity search, full-text search, and SQL queries all within a single platform.
*   **Multimodal Data Support**: Effortlessly store, query, and filter vectors, metadata, and diverse data types including text, images, videos, and point clouds.
*   **Advanced Features**: Benefit from zero-copy data access, automatic versioning for data management, and GPU acceleration for vector index building.

### **Key Advantages**

*   **Open Source & Local**: Enjoy the freedom of a 100% open-source solution that runs locally or in your cloud environment, eliminating vendor lock-in.
*   **Cloud & Enterprise Ready**: Leverage production-scale vector search without the need for server management, ensuring complete data sovereignty and security.
*   **Built on the Lance Columnar Format**: Built on the Lance columnar format for efficient storage and analytics.
*   **Seamless Integrations**: Integrate easily with Python, Node.js, Rust, and REST APIs. Native support for Python and JavaScript/Typescript.
*   **Extensive Ecosystem**: Compatible with popular tools such as [LangChain ü¶úÔ∏èüîó](https://python.langchain.com/docs/integrations/vectorstores/lancedb/), [LlamaIndex ü¶ô](https://gpt-index.readthedocs.io/en/latest/examples/vector_stores/LanceDBIndexDemo.html), Apache-Arrow, Pandas, Polars, DuckDB and more on the way.
