# LanceDB: The Open Source AI-Native Data Platform for Multimodal Data

**LanceDB is a fast, scalable, and production-ready vector database designed for AI/ML applications, enabling lightning-fast vector search and multimodal data management.**

[**Get Started with LanceDB**](https://lancedb.github.io/lancedb/basic/) | [**Explore the Documentation**](https://lancedb.github.io/lancedb/) | [**View Tutorials and Recipes**](https://github.com/lancedb/vectordb-recipes/tree/main) | [**Contribute to LanceDB**](https://github.com/lancedb/lancedb/graphs/contributors)

<br>

## Key Features of LanceDB:

*   **Blazing-Fast Vector Search:** Search billions of vectors in milliseconds with state-of-the-art indexing, powered by the Lance columnar format.
*   **Comprehensive Search Capabilities:** Supports vector similarity search, full-text search, and SQL queries, offering versatile data exploration.
*   **Multimodal Data Support:** Store, query, and filter vectors alongside metadata and multimodal data types like text, images, videos, and more.
*   **Advanced Features for Production:** Includes zero-copy data access, automatic versioning for data management, and GPU support for vector index building.
*   **Open Source & Cloud Options:** Enjoy the flexibility of a 100% open-source solution for local use or cloud-based production scale with complete data sovereignty.
*   **Seamless Integrations:** Native Python, JavaScript/Typescript, Rust, and REST APIs ensure easy integration into your existing workflows. ##  Installation & SDKs

Install LanceDB with ease:
```bash
# Python
pip install lancedb

# Typescript
npm install lancedb

# Rust
cargo add lancedb
```
