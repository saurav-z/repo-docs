# Download Google Chrome Offline Installers for Windows

**Need to install Google Chrome without an internet connection?** This repository provides a comprehensive archive of offline installers for various Chrome versions and architectures.  Find the latest versions of Chrome for Windows right here.

[Visit the original repository for more details.](https://github.com/Bush2021/chrome_installer)

**Key Features:**

*   **Offline Installers:** Download and install Chrome without needing an active internet connection.
*   **Multiple Architectures:**  Installers available for x86, x64, and ARM64 architectures.
*   **Different Channels:** Access installers for Stable, Beta, Dev, and Canary channels.
*   **Checksums Provided:** Verify the integrity of your downloads with SHA256 checksums.
*   **Version Details:**  Each installer includes the latest version, file size, and download link.
