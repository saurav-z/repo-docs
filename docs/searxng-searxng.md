# SearXNG: A Privacy-Respecting, Hackable Metasearch Engine

SearXNG is a powerful and customizable metasearch engine that prioritizes your privacy. ([Back to the original repo](https://github.com/searxng/searxng))

## Key Features

*   **Privacy-Focused:** SearXNG does not track your searches, ensuring your search history remains private.
*   **Open Source & Hackable:**  Built on open-source principles, allowing you to customize and contribute to the project.
*   **Metasearch Capabilities:** Searches multiple search engines simultaneously for comprehensive results.
*   **Customizable:**  Configure and tailor SearXNG to your specific needs and preferences.
*   **Easy Installation:** Available via Docker, installation scripts, and step-by-step instructions.
*   **Active Community:**  Engage with the SearXNG community on IRC and Matrix for support and discussions.
*   **Multi-Architecture Docker Images:** Supports x86-64, ARM64 and ARM/v7 architectures.
*   **Translations:** Help translate SearXNG into multiple languages via Weblate.
*   **GitHub Codespaces Support:** Easily contribute code directly from your browser.
