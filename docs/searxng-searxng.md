# SearXNG: Privacy-Respecting, Hackable Metasearch Engine

SearXNG is a free, open-source metasearch engine that empowers you to search the web without being tracked.  [Learn more on GitHub](https://github.com/searxng/searxng).

## Key Features:

*   **Privacy-Focused:** SearXNG does not track your searches or personal information.
*   **Open Source:** Freely available and customizable under the AGPL-3.0-or-later license.
*   **Metasearch:** Aggregates results from various search engines, providing a comprehensive view.
*   **Hackable & Customizable:** Modify and adapt SearXNG to your specific needs.
*   **User-Friendly:** Easy to install, configure and use.
*   **Multiple Instances:** Ready-to-use instances available on [Searx.space](https://searx.space).

## Getting Started

### Installation

*   **Docker:** Well-maintained Docker image available, including ARM64 and ARM/v7 architectures. See [SearxNG-docker](https://github.com/searxng/searxng-docker)
*   **Installation Scripts:** Up-to-date installation scripts are available. See [Installation Scripts](https://docs.searxng.org/admin/installation-scripts.html)
*   **Step-by-Step Instructions:** Detailed instructions for individual setup are available [here](https://docs.searxng.org/admin/installation-searxng.html).

## GitHub Codespaces

Contribute directly from your browser using [GitHub Codespaces](https://docs.github.com/en/codespaces/overview):

1.  Fork the repository.
2.  Click the "Code" button, then select the "Codespaces" tab.
3.  Click "Create codespace on master".
4.  VSCode will launch in your browser.
5.  Wait for `git pull && make install` to complete.
6.  Start SearXNG with `make run` or `Ctrl+Shift+B`.  You have [120 hours per month](https://github.com/settings/billing).
