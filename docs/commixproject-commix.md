# Commix: Automated Command Injection Exploitation Tool

**Commix** is a powerful, open-source penetration testing tool designed to automatically detect and exploit command injection vulnerabilities. ([See the original repo](https://github.com/commixproject/commix))

![Screenshot](https://commixproject.com/images/background.png)

Explore Commix's capabilities with the [screenshot collection](https://github.com/commixproject/commix/wiki/Screenshots).

## Key Features

*   **Automated Vulnerability Detection:** Quickly identifies command injection flaws.
*   **Exploitation Capabilities:** Automates the process of exploiting command injection vulnerabilities.
*   **Open Source & Free:** Leverage the power of a community-driven security tool.
*   **Cross-Platform Compatibility:** Works on any platform where Python runs.
*   **User-Friendly Interface:** Easy to use and understand, even for beginners.

## Installation

Get started with Commix by cloning the official Git repository:

```bash
$ git clone https://github.com/commixproject/commix.git commix
```

Alternatively, download the latest [tarball](https://github.com/commixproject/commix/tarball/master) or [zipball](https://github.com/commixproject/commix/zipball/master).

**Note:** Commix requires **[Python](http://www.python.org/download/)** (version **2.6**, **2.7**, or **3.x**).

## Usage

Find a comprehensive list of options by running:

```bash
$ python commix.py -h
```
