# Commix: Automate Command Injection Exploitation (Penetration Testing Tool)

**Commix** (short for **Comm**and **i**njection e**x**ploiter) is an open-source penetration testing tool that automatically detects and exploits command injection vulnerabilities in web applications.  ([Original Repository](https://github.com/commixproject/commix))

![Screenshot](https://commixproject.com/images/background.png)

Explore the [screenshots](https://github.com/commixproject/commix/wiki/Screenshots) to see Commix in action.

## Key Features

*   **Automated Exploitation:**  Identifies and exploits command injection vulnerabilities with minimal user input.
*   **Open Source:** Freely available and customizable for security professionals and enthusiasts.
*   **Versatile:** Supports various operating systems and web application configurations.
*   **User-Friendly:** Designed for ease of use with a clear command-line interface.
*   **Comprehensive:**  Includes features for vulnerability detection, exploitation, and post-exploitation.

## Installation

Get started by cloning the repository:

```bash
git clone https://github.com/commixproject/commix.git commix
```

Alternatively, download the latest [tarball](https://github.com/commixproject/commix/tarball/master) or [zipball](https://github.com/commixproject/commix/zipball/master).

*Note: Requires **Python** (version **2.6**, **2.7** or **3.x**).*

## Usage

Get a list of all options:
