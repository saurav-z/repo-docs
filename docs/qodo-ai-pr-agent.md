## Key Features

*   **AI-Powered Review:** Get automated code reviews, identifying potential issues and suggesting improvements.
*   **Automated Tasks:** Automate routine PR tasks such as generating descriptions, updating changelogs, and adding documentation.
*   **Multi-Platform Support:** Works seamlessly with GitHub, GitLab, Bitbucket, and Azure DevOps.
*   **Flexible Integration:** Use PR-Agent as a GitHub Action, CLI tool, or through a Chrome extension.
*   **Customizable Tools:** Tailor PR-Agent's behavior using configuration files for specific project needs.
*   **Qodo Merge Integration:**  Access advanced features, priority support, and enhanced privacy with Qodo Merge.

## Why Use PR-Agent?

PR-Agent is designed to streamline your pull request workflow, providing quick, affordable, and practical AI-driven assistance. Unlike other tools, PR-Agent emphasizes:

*   **Efficiency:** Get rapid feedback with single LLM calls.
*   **Effective PR Handling:** Leverage PR Compression Strategy to efficiently handle both short and long PRs.
*   **Modularity:** Easily customize tools using a JSON prompting strategy.
*   **Wide Compatibility:** Supports multiple Git providers, usage methods (CLI, Action, App, Docker, etc.), and language models (GPT, Claude, Deepseek, etc.)
